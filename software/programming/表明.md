# 表明

Assertion

あるコードが実行される時に満たされるべき条件を記述して実行時にチェックする仕組みのこと。

実際の処理の前に前提条件を明記することで、余計な処理に入ってからエラーで落ちるのを避けられたり、コードが読みやすくなったりする。

単に `if` でチェックして抜けたり、専用の `assert` 文やライブラリがあったりする。

Swift の「ガード節」なども表明を助ける機構の一種。

PHP の場合、production 実行時に `assert()` を削除する設定もできる。
