1. 公開されているプラグインを導入する
2. プロジェクトのフォルダを作って、作りたいものを相談して、試しに作ってみる
3. 相談した内容から`/init`で`./CLAUDE.md`を作る
4. 共通の`~/.claude/CLAUDE.md`を育てる
5. 自前のスキルやサブエージェントを作る

まずはこのような手順で進めるといい。

## セッションごとのコツ

- 空のフォルダは`git init`してから始める (セッションが始まってから`git init`しても権限エラーで書き込めない)
- 空のフォルダはやりたいことを話してから`/init`する

## その他

いきなりClaude Codeで相談を始める場合は適当に仮名をつけたフォルダで作業を始めるといい。具体的な名前をつけても話しているうちに考えていた方向性と違ってくることもある。
