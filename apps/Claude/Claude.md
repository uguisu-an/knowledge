- 並列で3-5個同時に回せ
- 複雑タスクはプランモードから
- CLAUDE.mdに全てを記録
- スキルを自作してgit管理
- バグ修正は丸投げでOK
- プロンプトをレベルアップ
- ターミナル環境を整えろ
- サブエージェントを使え
- データ分析もClaudeで
- 学習ツールとして活用

何ができるか、アイデアを思いついたら、Claude自身に聞いてみよう。
例えば「YouTubeの動画を要約できますか？」とか。

「あなたならどうする？」みたいな質問で手順を聞いて、「ここをこうしてほしい」みたいにブラッシュアップしてから実行するのもいい。

Claude Codeを使うとローカルのファイルにもアクセスできる。
ターミナルでClaudeを起動して、あとはチャットと同じように指示するだけ。

Google検索とノートを使ってやるようなことは大体できる。

Web版のデフォルトはSonnetだが、コーディングはOpus (thinking) が強い。

Claudeが間違ったことをするたびに`CLAUDE.md`に追記する。

Agent Teamは複数のエージェントが一気に動くので、トークンの消費量が多い。何にでも使っていい機能ではない。

Claudeは呼び出し制限が厳しいので、質問するだけならChatGPTやGeminiに投げたほうがいい。Claudeはコーディングなどのタスクに使う。

Proプランは年に220ドル、2026年で36000円くらい。
CodeやCoworkの使い道がはっきりあれば安いんだけど、たまに使うくらいだと高い。
