## 最初のモデル

写真系:

- Flux.1 Schnell: 高速
- Flux.2 Klein 4B: 高速、メモリ16GB程度から利用できる

## 最初の設定

- LoRA: 無効
- コントロール: 無効
- シード: -1 (ランダム)

## モデルの選択

- SD 1.5系: 旧世代だが軽量
- SDXL系: イラスト系の既存リソースが充実
- Flux系: 現在の最高峰

とりあえず写実系はFluxで、イラスト系はSDXLで始めるといい。
