# フォルダ構成

| フォルダ    | 概要                       |
| ----------- | -------------------------- |
| `templates` | そのまま使えるテンプレート |
| `docs`      | 詳細な資料                 |
| `assets`    | 画像とか                   |
| `misc`      | 未整理のメモ               |

`docs` の中が厚いところは語彙をまとめるための `reference` とそれぞれの扱いをまとめるための `how-to` に分ける形式のほうがいいかも。
あと `docs` 自体もフォルダ分けが細かくなってきたのでもう少しまとめた方がいいかも。
