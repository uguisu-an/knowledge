# Programming Paradigms

「プログラミングパラダイム」はプログラミングの歴史の中で変化した大きな潮流・思想のこと。

1. 構造化プログラミング (Structured Programming: SP)
2. オブジェクト指向プログラミング (Object-Oriented Programming: OOP)
3. 関数型プログラミング (Functional Programming: FP)

の 3 つがある。今のところ 3 つしかない。
