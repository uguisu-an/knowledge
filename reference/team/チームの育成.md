# チームの育成

チームの核は「自己管理」だ。

個人が自らチームであろうとして、自らをマネジメントする。

自分で仕事に必要な作業を判断して、いらないものを切り捨てる。

自ら情報を集めて、連携する。

個人の専門的な能力はある程度持っておかないと、自己管理と連携を育てるレベルになるまで時間がかかる。

あくまでも専門性はある程度持っている前提で、自己管理と連携を積み重ねる。

専門性の低い、なんの役にも立たない人は、チームに入れる前に個人として教育、研修する。

コミュニケーションの改善よりも前に能力密度を高めないといけない。
能力の低い人を排除するか、一から養成するか。
常識を身に付けるための教育はどっちみち必要。
