# チームの育成

チームの核は「自己管理」だ。

個人が自らチームであろうとして、自らをマネジメントする。

自分で仕事に必要な作業を判断して、いらないものを切り捨てる。

自ら情報を集めて、連携する。

個人の専門的な能力はある程度持っておかないと、自己管理と連携を育てるレベルになるまで時間がかかる。

あくまでも専門性はある程度持っている前提で、自己管理と連携を積み重ねる。

専門性の低い、なんの役にも立たない人は、チームに入れる前に個人として教育、研修する。

コミュニケーションの改善よりも前に能力密度を高めないといけない。
能力の低い人を排除するか、一から養成するか。
常識を身に付けるための教育はどっちみち必要。

まずは組織を作って、ある仕事に複数人で当たるようにするところから。
一人でしか仕事できない人をいきなり集めてもうまくいかない。
まずは他人と一緒に仕事することを学ぶ。

1. 個人の速さを高める
2. 大きな仕事のために人数を増やす
3. 個人の速さが落ちる
4. チームとしての施策で個人の速さを戻す
5. 大きな仕事をしながら個人の速さも落ちない＝最高のパフォーマンス

チームは「集める」んじゃなくて「分ける」形にする。
まず組織で仕事できる人、コミュニケーションできる人じゃないとチームにしてもうまくいかない。
