# 恋人ができるまで

## 成果目標

恋人ができるために、以下の制約を満たす。

1. [知人ができる](./知人ができるまで.md)
2. [友人ができる](./友人ができるまで.md)
3. セックスしたくなる
4. 相手もセックスしていいと思っている（また、それに気づく）
5. セックスに誘って、または誘われて合意する

## 行動目標

段階的にスキンシップを進める。

相手から誘われるのを待つだけでなく、自分からも誘う。

## 備考

赤の他人 → 知人 → 友人 → 恋人のように発展する。

知人や赤の他人でも消極的に愛することはできる。
相手の存在を認めて、殺さない。消極的な愛はコストがかからないので無限。
しかし、積極的な愛や好意、恋は有限だ。
