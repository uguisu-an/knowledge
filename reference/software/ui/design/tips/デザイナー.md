# デザイナー

[どういうデザイナーとだと仕事しやすいか - Konifar's ZATSU](https://konifar-zatsu.hatenadiary.jp/entry/2017/11/14/121512)

実装に依存する部分は、デザイナーにも知識として必要になる。
例えば、木製品であれば木の重さや質感、油絵であれば油絵の具の照りや質感、など。
油絵具に触ったことがないと、油絵の監修をしようとしてもうまくいかない。

実現不可能なコンセプトデザインを吐き出すのもデザイナーとしては必要だけど、それらのデザイナーは実際にどの程度が実現可能なのか知らないわけではない。
知った上で、知らないふりもできる。
むしろ知らないふりは知らないとできない。
