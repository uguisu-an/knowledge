# 主な拡張機能

| 名称                            | 説明                                 |
| ------------------------------- | ------------------------------------ |
| Advanced new file               | コマンドパレットのファイル作成の拡張 |
| Auto Rename Tag                 | HTML タグの編集を補助する            |
| Better Comments                 | 特殊なコメントに色をつける           |
| EditorConfig for VS Code        | editorconfig の設定を反映する        |
| Git History                     | 編集履歴の参照                       |
| GitLens                         | Git 機能の拡張                       |
| GitHub Pull Requests and Issues | GitHub 連携の補助                    |
| Live Share                      | リモートで同時編集                   |
| Increment Selection             | 採番                                 |
| Laravel Blade formatter         | Blade のフォーマッタ                 |
| Partial Diff                    | 文字列比較                           |
| Path Intellisense               | パスの入力補助                       |
| PHP Constructor                 | PHP 用の入力補助                     |
| PHP Getters & Setters           | PHP 用の入力補助                     |
| PHP Intelephense                | PHP のサポート                       |
| PHP Namespace Resolver          | PHP 用の入力補助                     |
| PHP Refactor Tool               | PHP 用の編集補助                     |
| PHPUnit                         | PHPUnit の呼び出し補助               |
| Prettier                        | JS や CSS のフォーマッタ             |
| Rainbow CSV                     | CSV を列ごとに色分けする             |
| Settings Sync                   | 設定をクラウドに保存する             |
| Vetur                           | Vue のサポート                       |
| YAML                            | YAML のサポート                      |
