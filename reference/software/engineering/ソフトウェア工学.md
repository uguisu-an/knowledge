# ソフトウェア工学

ソフトウェアを対象とした工学の応用分野。

設計や実装などの開発、検証に加えて、運用、保守まで含まれる。
分析も含むことがある。

-   computer (コンピュータ)
    -   software (ソフトウェア)
    -   science (コンピュータ科学)
-   development (開発)
    -   design (設計)
    -   prototyping (プロトタイピング)
    -   programming (プログラミング)
    -   coding (コーディング)
    -   building (ビルド)
    -   testing (テスト)
-   operation (運用)
    -   infrastructure (インフラ)
    -   deployment (デプロイ)
-   maintenance (保守)
    -   monitoring (監視)

ソフトウェアのような情報技術には量産の工程がないのも特徴。
