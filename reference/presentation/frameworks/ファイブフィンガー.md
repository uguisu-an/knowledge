# Five Fingers

片手で今の仕事の状況を表明する。

1. 自分ではどうしようもないので相談したい
2. 今日のタスクは終わらないので助けが必要
3. 今日のタスクは終わりそうだが自信はない
4. 今日のタスクは終わる自信がある
5. 今日のタスクは終わって他の人を助けられる

スクラムの朝会などに導入するといい。
