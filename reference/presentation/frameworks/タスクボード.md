# タスクボード

カンバンとも呼ばれる。

工程ごとに並べられた横一列のリストで、ある課題がどの工程まで進んでいるかを見せる。

たとえば、以下のようなボードが考えられる。

| 未着手 | 工程 A | 工程 B | 工程 C | 出荷   |
| ------ | ------ | ------ | ------ | ------ |
|        |        |        |        | 課題 A |
|        |        |        | 課題 B |        |
|        | 課題 C |        |        |        |
| 課題 D |        |        |        |        |

## 一人ではイマイチ

タスクボードは複数人で使うものなので、一人で使ってもイマイチ。
他の人がやっていること、やったことが見えるようになるのが大事。
