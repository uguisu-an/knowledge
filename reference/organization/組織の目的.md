# 組織の目的

組織は同じ目的、仕事のためにある。
それを共有できず、真剣に向き合わない人は排除すべき。

大きな仕事のためにある。

組織では大きな仕事をするためにパフォーマンスを落とすことがある。
とにかく大きな仕事をする方が重要。

一人で仕事してると「ここどうしてこうなってるの？」ってとこも誰かに言われるまで気づかないことがある。
盲点を補い合うのも組織体制を作る理由。

仕事が変われば、組織も変えないといけない。

ビジョン。

組織のトップがやるべきことは、組織の目的を決めること、体制と文化を作ること。
細かいところはさらに細かい内部組織に任せたらいい。

組織の中でさらに細かい目的を持った小さな組織を持つこともある。

活動や事業のため、教育や研修のため、仕事の割り振りのため、など。

新入社員が教育部にしばらく所属するのも一旦教育目的で分けている。
