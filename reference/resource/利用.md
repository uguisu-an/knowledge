# 利用 Use

"use" には「働かせる」「仕事させる」意味がある。

資源管理は権限管理。
ある資源を利用できるかどうかを管理する。

本来利用できてはいけない資源を利用されると「不正利用」になる。
これを防ぐのはセキュリティ。

ICT では情報というリソースを主に扱う。

利用できる (available) かどうかの状態を持つ。
実際に結果を出せる能力の問題と、権限の問題がある。
development と engagement にも関わりがある。
特に engagement は、人間を資源として捉えたら、権限を得るための活動になる。

利用するだけなら所有しなくてもいい。

利用者は User という。

資源をどこで利用するのか割り当てるのが administration.

経済、生産と消費も「利用」と関連がある。

より多くの資源を得るために手持ちの資源を使うのが投資。
