# E2E CRUD Test Example

ユーザー管理機能をテストする例。

## 共通

運営としてログインしている。

ユーザーの一覧ページにいる。

## Create

「新規作成」ボタンをクリックする。

→ ユーザーの新規作成ページに移動する。

名前を入力する。

「保存」ボタンをクリックする。

→ ユーザーの一覧ページに移動する。

## Read

→ 一覧の一番上に ユーザー 1 の名前が表示される。

## Update

ユーザー 1 の名前のリンクをクリックする。

→ ユーザーの編集ページに移動する。

→ ユーザー 1 の名前が入力されている。

新しい名前を入力する。

「保存」ボタンをクリックする。

→ ユーザーの一覧ページに移動する。

→ 一覧に ユーザー 1 の新しい名前が表示される。

## Delete

ユーザー 1 の「削除」ボタンをクリックする。

→ ユーザーの一覧ページに移動する。

→ 一覧の中に ユーザー 1 の名前が表示されない。
