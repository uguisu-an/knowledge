## その他

五目並べは相手の手が伸びないようにしつつ自分の手を伸ばせる手が強い。

以下のような四角や菱形の形が作れると強い。防がれないまま x の位置に置くと勝ち。

A:

```
o_o
_x_
o_o
```

B:

```
_o_
oxo
_o_
```

五目並べは両端が塞がってない3や飛び3を一度に2組作るのが基本。
つまり、同じ目に置くと2組の3になる2の2をまずは用意する。
応用としては、片側が塞がれている4や飛び4でもいい。

三三や四四は詰みの形なので、この形になったらもう手遅れ。
詰みの前の段階、詰めろで止めないと遅い。

五目並べを禁じ手ありで遊ぶ時は三三や四四が禁じられるので、基本的に三四で作らないといけない。

五目並べの基本はダイヤの形。フリーのダイヤを作って真ん中に置くと3, 3になる。
ただ、この形で妨害されずに3, 3が完成することはまず無いので、応用が必要。2, 2の形はダイヤをずらした形で特に重要。

五目並べは詰めろの応酬。相手が正しい手を打たないと負け確になる状態を維持してながら、詰みに持っていく。
逆に、守る側は詰めろ逃れの詰めろで切り返す。

後手はまず守りで、どこかで切り返さないといけない。

五目並べの守りでは、ただ受けるだけじゃなくて、受けながら相手も絶対に受けないといけない形を作って、攻めの手番を奪う。逆に、攻める時は手番を奪われる攻めはダメ。

五目並べは3個で2、1の形が攻めの基本形。次の一手で3、2になれば、相手に3を止められてももう一つ3が作れる。うまく連鎖すればそのまま連続で相手の手をコントロールできる。そうなると、相手は2、1の段階で早めに受けることになりやすいので、攻めを続けられる。

五目並べは絶対に止めないといけない形を一つずつ覚えていって、それができたら止める、自分の手番ではそれを作れないか狙う。まずはこれが初心者の基本。ミス待ちで勝つ。最低限が、3個並んだ形。1つ飛ばしで3個の場合も見逃さないようにする。
あとは、基本的な3個で2、1の形を狙っていく。攻めでも守りでも。
自分が絶対に止めないといけない形を作り続ければ、相手は限られた手しか選べない。それを組み合わせて、より長手数の詰みの形を狙えるようにしていくのが次の段階。逆に守りでもより先を読む。

五目並べの攻めは3個で2,2を作り続けるのが基本。相手は基本的に2手以内で受け続けないといけない。
その中で良い形になったら4個で3,2を作ったりして相手の守りを崩す。

守りでは、まず相手の打ちたいところを洗い出して、危険度で優先順位をつける。最も優先順位が高いところに打てばいい。どこにも急いで打たなくていいなら攻めの手を打つ。

攻めるか受けるかは、速度と相談。自分の方が早く勝てるなら自分の手を優先して攻めていい。そうでないならできるだけ攻めにも繋がる手で受ける。
