# Kimberly

攻めでリスクをとって、反撃の可能性があっても攻めを切らさないのが大事。
不利なところから無理やり読み合ったりすることもある。

差し合いは強くも弱くもないけど、ラッシュは強い。

キンバリーはあまり差し合いが強くないので、できるだけ付き合わない。
