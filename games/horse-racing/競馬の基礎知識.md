# 競馬の基礎知識

馬券を選ぶときに前提として持っておきたい知識を体系的にまとめたもの。
個々の券種・データの「意味」と「予想への効き方」を一覧できることを目的とする。
具体的な買い方の方針は別ノート（`馬券を選ぶ.md`）にまとめる。

## 全体像

馬券選びは、おおまかに次の流れで考える。

1. **レースの条件を知る**（クラス・距離・芝/ダート・コース・馬場）
2. **出走馬を評価する**（成績・タイム・脚質・適性・騎手）
3. **展開を読む**（誰が逃げ、どんなペースになるか）
4. **オッズと照らす**（市場の評価＝オッズと自分の評価のズレを探す）
5. **券種と点数を決める**（リスクとリターンの設計）

「強い馬を当てる」のではなく「市場が過小評価している馬・買い目を見つける」ゲームだと捉えるのが基本。
的中率より回収率を追う。

## 馬券の種類（券種）

| 券種   | 的中条件                       | 点数の目安 | 払戻率 | 控除率 | 性格                          |
| ------ | ------------------------------ | ---------- | ------ | ------ | ----------------------------- |
| 単勝   | 1 着の 1 頭                    | 1          | 80%    | 20%    | 全ての基礎。まずこれを当てる  |
| 複勝   | 3 着以内の 1 頭                | 1          | 80%    | 20%    | 最も当たりやすい。低配当      |
| 枠連   | 1・2 着の 2 枠（順不同）       | 中         | 77.5%  | 22.5%  | 枠単位。同枠 2 頭でお得な場合 |
| 馬連   | 1・2 着の 2 頭（順不同）       | 中         | 77.5%  | 22.5%  | 軸が決まったら基本の組合せ    |
| ワイド | 3 着以内の 2 頭（順不同）      | 中         | 77.5%  | 22.5%  | 馬連より当たりやすい中配当    |
| 馬単   | 1・2 着の 2 頭（着順まで）     | 多         | 75%    | 25%    | 馬連の着順指定版              |
| 三連複 | 3 着以内の 3 頭（順不同）      | 多         | 75%    | 25%    | 高配当だが難度が上がる        |
| 三連単 | 1〜3 着の 3 頭（着順まで）     | 非常に多   | 72.5%  | 27.5%  | 最も高配当・最も難しい        |
| WIN5   | 指定 5 レースの 1 着を全て的中 | 膨大       | 70%    | 30%    | 宝くじ的。控除率最大          |

- **控除率（テラ銭）** は売上から胴元が取る割合。残り（払戻率）が当たった人に分配される。
  券種によって違い、**当てにくい券種ほど控除率が高い**＝長く買うほど不利になりやすい。
- **複勝とワイドの「3 着以内」は出走頭数で変わる**。出走 8 頭以上なら 3 着まで、5〜7 頭なら 2 着まで、
  4 頭以下は複勝・ワイドの発売がない。
- **応援馬券**は単勝＋複勝のセット券で、券種としては別物ではない。

## オッズ・人気・期待値

- **オッズ（倍率）= 払戻倍率**。単勝オッズ 10 倍なら、100 円が的中で約 1,000 円になる。
- **人気順**はオッズの低い順。市場がどの馬を勝つと見ているかの縮図。
- **オッズは確定ではなく変動する**。締切（発走）まで売れ方で動く（パリミュチュエル方式）。
- **市場が見込む勝率 ≒ 1 ÷ オッズ × 払戻率**。ざっくり「1 ÷ オッズ」を勝率の目安にする。

| 単勝オッズ | 市場が見込む勝率の目安 |
| ---------- | ---------------------- |
| 1.1 倍     | 約 90% 以上            |
| 2.0 倍     | 約 50% 以上            |
| 5.0 倍     | 約 20% 以上            |
| 10 倍      | 約 10% 以上            |

- **期待値 = 自分が見積もる勝率 × オッズ**。1 を超えれば長期的にプラス。
  「強いか」ではなく「オッズに対して強すぎるか／弱すぎるか」で判断する。
- 一番人気は期待されすぎていることが多い（オッズの割に勝率が見合わない）。
  逆に、人気でもなお市場の評価が足りないこともある。

## レースの条件

### クラス（条件）

勝つごとに上のクラスへ上がる。下から順に強くなる。

```
新馬 / 未勝利 → 1 勝クラス → 2 勝クラス → 3 勝クラス → オープン → 重賞（GⅢ < GⅡ < GⅠ）
                                                       └ リステッド(L) / オープン特別
```

- **収得賞金**でクラスが決まる。勝利で賞金が加算され、基準を超えると昇級。
- 重賞（GⅠ〜GⅢ）は「グレードレース」。リステッドは重賞に準じる格付け。
- **昇級初戦**は相手が強くなるため割引いて見る。逆に**格下げ・同級維持**は狙い目になりやすい。

### 距離（SMILE 区分）

平地は 1000〜3600m。国際基準の SMILE 区分が使われる。

| 区分                      | 距離        | 得意な馬の呼称 |
| ------------------------- | ----------- | -------------- |
| S（スプリント）           | 1000〜1300m | スプリンター   |
| M（マイル）               | 1301〜1899m | マイラー       |
| I（インターミディエイト） | 1900〜2100m | （中距離）     |
| L（ロング）               | 2101〜2700m | （中長距離）   |
| E（エクステンディッド）   | 2701m〜     | ステイヤー     |

- 一般的な言い方では、〜1400m=短距離、1600m=マイル、〜2000m=中距離、2400m〜=長距離。
- **距離適性**は最重要級。同じ馬でも得意距離以外では大きく割引く。
- 短距離は展開・駆け引きの余地が小さく、長距離ほどペースとスタミナが効く。

### 馬場（コース）

- **芝 / ダート（砂）**: 適性が大きく分かれる。芝向きの馬はダートで凡走しやすく、その逆も。
- **右回り / 左回り**: 得手不得手が出る馬がいる。
- **コース形態**: 直線の長さ、坂の有無、コーナーのきつさで有利な脚質が変わる。
- **馬場状態**: 良 → 稍重 → 重 → 不良 の順に時計（タイム）がかかる。
  - 雨・道悪（重・不良）は**荒れやすく予想が難しい**。
  - **道悪巧者**（重馬場に強い血統・馬）は妙味になる。
- **天候・気温・風**も影響するが、効くのは主に馬場状態と時計。

## 出馬表（出走表）の見方

| 項目             | 見るポイント                                              |
| ---------------- | --------------------------------------------------------- |
| 馬番 / 枠番      | 内外の有利不利、枠連の単位                                |
| 性齢             | 牡（オス）・牝（メス）・セン（去勢）と年齢                |
| 斤量（負担重量） | 背負う重さ。重いほど不利になりやすい（後述）              |
| 馬体重 / 前走差  | 増減が大きいと状態変化のサイン。極端な増減は割引          |
| 騎手             | 勝率・連対率・3 着内率。上位騎手は信頼度が上がる          |
| 過去成績         | 着順・着差・通過順位・走破タイム・上がり 3F               |
| 前走間隔         | 詰まりすぎ／空きすぎは割引材料。叩き 2 戦目で良化する馬も |
| 血統             | 父・母父で距離・芝ダート・道悪適性の傾向を推測            |
| 調教・パドック   | 直前の仕上がり。基本は過去成績で足り、極端な変化だけ確認  |

- **直近 1〜3 走の成績**を特に重視する。古い好走より直近の状態。
- パドックや調教は補助。雰囲気だけで買うなら複勝など低リスクに留める。

## 脚質と展開

レース中の位置取り・走り方のタイプ。展開の有利不利を左右する。

| 脚質 | 位置取り           | 特徴                                       |
| ---- | ------------------ | ------------------------------------------ |
| 逃げ | 先頭               | ペースを自分で作れる。スローだと特に有利   |
| 先行 | 先頭から 4〜5 番手 | 馬群を捌きやすく、ペースに左右されにくい   |
| 差し | 中団〜やや後方     | 脚を溜め、直線で差し込む。展開待ちの面も   |
| 追込 | 後方               | 直線勝負一本。ハイペース・上がり勝負で台頭 |

- **基本は前にいる馬（逃げ・先行）が有利**。自分でペースを作れるため。
- **ペースと脚質の相性**:
  - スローペース → 前残り（逃げ・先行有利）
  - ハイペース → 前が垂れて差し・追込が届きやすい
- **逃げ馬が何頭いるか**で展開が読める。逃げ不在ならスロー、複数いれば速くなりやすい。
- 枠順・馬場・コース形態と合わせて、「誰がどこを通って、どこで脚を使うか」を想像する。

## 馬を評価する主要データ

- **走破タイム**: 速さの基本指標。ただし馬場・展開で変わるので絶対値だけでは比べにくい。
- **上がり 3 ハロン（上がり 3F）**: ゴール前 600m のタイム。**末脚の鋭さ**を示す重要指標。
  「上がり最速」を頻繁に記録する馬は決め手がある。
- **着差**: 1 着馬とのタイム差・馬身差。僅差の負けは評価できる。
- **通過順位**: 各コーナーでの位置取り。脚質と展開の答え合わせに使う。
- **着順**: 1〜3 着か着外か。ただし相手やペースを踏まえて中身を見る。
- **基準タイムでの足切り**: 過去に 3 着以内の基準タイムを出せていない馬は思い切って外す。

## 斤量（負担重量）

馬が背負う重量。重いほど不利になりやすい。決め方で 4 タイプ。

| 方式         | 決め方                                   |
| ------------ | ---------------------------------------- |
| 馬齢（定量） | 年齢・性別で一律。多くの GⅠ・重賞で採用  |
| 別定         | 過去の実績（賞金・勝ち鞍）で加増         |
| ハンデ       | ハンデキャッパーが力量差を埋めるよう設定 |
| 定量         | レース規定で固定                         |

- **牝馬は減量**: 3 歳以上は牡馬より 2kg 軽い（2 歳 10〜12 月は 1kg 減）。
- 近年（2023〜2024）は基礎重量が引き上げられ、**GⅠ 定量は 58kg 基準**になった。
- ハンデ戦は実力上位馬ほど重く、**荒れやすい**。

## 主要レース体系（覚えておくと格が分かる）

- **クラシック（3 歳限定の伝統 5 競走）**
  - 牡馬三冠: 皐月賞 → 日本ダービー（東京優駿）→ 菊花賞
  - 牝馬三冠: 桜花賞 → オークス（優駿牝馬）→ 秋華賞
- **秋古馬三冠**: 天皇賞（秋）→ ジャパンカップ → 有馬記念
- 大レース（GⅠ）はトップ級が集まり**実力が拮抗するため荒れやすい**。
  見る分にはレベルが高く面白いが、「GⅠ だから儲かる」わけではない。

## 馬券選びの要点（考え方）

- **まず単勝・複勝を当てられること**。1 頭も当てられないのに、2 頭・3 頭は当てられない。
- **軸が絞れないレースは買わない**。軸が絞れて初めて連系・3 連系を検討する。
- **的中率より回収率**。1,000 円買って 1,000 円以上戻るのが第一目標。
- **買い目を増やしすぎない**。「当てる」のと同じくらい「無駄な買い目を削る」のが重要。
  当たっているのに収支マイナスは一番もったいない。
- ボックス・流しは点数（＝ハズレ馬券）が増えやすく、基本はコスパが悪い。
- 雰囲気・推しで買うなら低リスク（複勝・ワイド・少額）に留める。

## タイムへの影響

- 馬場ごとの平均タイム (https://w.atwiki.jp/monosashi-keiba/pages/17.html)
  - 東京と新潟は平均タイムが少し速い (長い直線、緩やかなコーナー、ゆるい坂)

## 有利不利

- 体が小さいのは不利
  - 4-5歳馬は有利 (平均30kgくらい大きい)
  - 牝は不利 (平均20kgくらい小さい)
- 体が重い (太っている) のは不利
- 騎手が弱いのは不利
- 短い距離しか経験がないのは不利
