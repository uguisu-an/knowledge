# Grooming

清潔感。身だしなみ。

清潔感がないと、親しくなりたくない。

清潔感がないと、病気や貧困を連想させる。

## 清潔感がなくなっている要因

- 不健康に感じる
  - 肌が乾燥している
    - 肌がテカっている
    - 肌が荒れている
  - 毛穴が目立つ
    - 毛穴が開いている
    - 毛穴が詰まっている
    - 皮膚がたるんでいる
  - 髪が乾燥している
  - 髪が伸びている
  - 髪が整えられていない
  - 顔の部品が歪んでいる
    - 形が悪い（縦のバランスが悪い）
    - 左右対称になっていない（横のバランスが悪い）
  - 臭い
    - 口が臭い
      - 口が乾燥している
      - 舌が汚れている
      - 歯が汚れている
      - 内臓から匂っている
    - 体が臭い
      - わきが
    - 髪が臭い
      - 生乾きで放置した
    - 服が臭い
      - 生乾きで放置した
  - 病気っぽい
    - 赤ら顔でぼーっとしている
    - 咳やくしゃみをしている
- ずぼら
  - 服が汚れている
  - 服にシワがついている

## その他

掃除と服装は清潔感を与える意味で似ている。衛生に関連する。

清潔な髪と肌と服があれば、最低限身だしなみには気を遣ってるように見える。

垢抜けた印象を与える場合、髪型を遊ぶ、肌が綺麗、ムダ毛や肌荒れを隠す、シュッとした明るい服、フェイスラインが綺麗、とか。

優先度は、以下の通り。

1. 美容室で髪を切って、髭は剃る
2. 肌をケアして、肌荒れを隠す
3. ユニクロなどでマネキンの服を買ってそのまま着る
4. 太り過ぎ、痩せ過ぎは避ける
