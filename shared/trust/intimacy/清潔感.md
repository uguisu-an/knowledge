# Grooming

清潔感。身だしなみ。

清潔感がないと、親しくなりたくない。

清潔感がないと、病気や貧困を連想させる。

清潔感は、「綺麗」というよりも「怪しくない」こと。

## 清潔感がなくなっている要因

- 遺伝子的に悪く感じる
  - 顔が歪んでいる
    - 形が悪い（縦のバランスが悪い）
    - 左右対称になっていない（横のバランスが悪い）
- 不健康に感じる
  - 老いを感じる
    - 肌が乾燥している
      - 肌がテカっている
      - 肌が荒れている
    - 毛穴が目立つ
      - 毛穴が開いている
      - 毛穴が詰まっている
    - 皮膚がたるんでいる
    - 髪が乾燥している
  - 太りすぎている
  - 痩せすぎている
  - 病気っぽい
    - 赤ら顔でぼーっとしている
    - 咳やくしゃみをしている
- 不潔に感じる
  - 臭い
    - 口が臭い
      - 口が乾燥している
      - 歯や舌が汚れている
      - 内臓から匂っている
    - 体が臭い
      - わきが
    - 髪や服が臭い
      - 生乾きで放置した
  - 髪が伸びている
  - 髪がべたついている
  - 髪が整えられていない
  - ひげが伸びている
  - 服が汚れている
  - 服が傷んでいる
  - 服がしわくちゃになっている
- 常識的でない
  - 常識的でない言動をしている
  - 常識的でない服装をしている

## その他

掃除と服装は清潔感を与える意味で似ている。衛生に関連する。

清潔な髪と肌と服があれば、最低限身だしなみには気を遣ってるように見える。

垢抜けた印象を与える場合、髪型を遊ぶ、肌が綺麗、ムダ毛や肌荒れを隠す、シュッとした明るい服、フェイスラインが綺麗、とか。

優先度は、以下の通り。

1. 美容室で髪を切って、髭は剃る
2. 肌をケアして、肌荒れを隠す
3. ユニクロなどでマネキンの服を買ってそのまま着る
4. 太り過ぎ、痩せ過ぎは避ける

格下に見られないのが大事。

シャツはできるだけこまめに買い替えた方が色や匂いもないし襟も伸びてないので綺麗。古いのは買った店のリサイクルに出せばいいし。

清潔感があるだけじゃなくて、期待感があるといい。

「清潔感がない」とか、「生理的に無理」っていうのは「抱かれたくない」ってこと。
まぁ、特にひどい場合は接触がなくても病気をうつされそうだから死んでほしいって意味だけど、そこまではなかなかいない。

肌のテカリも実は乾燥から来ている。

不健康に見えると性的魅力も清潔感もない。

性的魅力と近い。

自分の清潔感を異性に厳しく採点してもらうのも話題としていいかも。ちょい深めの話題。
