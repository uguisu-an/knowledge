# 個人の印象

人を覚えるためのメモ。各領域を「特徴（観察できる客観的な事実）」と
「印象（そこから受ける主観）」の対で見る。特徴は観察のため、印象はその人をざっくり掴むため。
印象は対になる軸（A / B）でどちら寄りかを見るとざっくり分類しやすい。
観察項目の細目は 人を見る要素.md（外見）／人の要素.md（基本情報・各側面）を参照。

## 全体・第一印象

ひと言でどんな人か。領域をまたいだ総合的な印象。

- 特徴: 第一印象、ぱっと見の雰囲気、ひと言でいうと（キャッチコピー）
- 印象:
  - 好き / 嫌い、好印象 / 悪印象
  - 温かい / 冷たい
  - 有能 / 無能
  - すごい / 普通
  - 強い / 弱い
  - 敵に回したくない / くみしやすい
  - 美しい / 醜い
  - 怪しい / 信頼できる
  - 近寄りがたい・圧がある / 親しみやすい
  - 近づきたい / 距離を置きたい
  - 覚えやすい / 印象が薄い
  - 守りたい / 心配
  - 人気がある / 人気がない
  - 嫌悪感がある・生理的に無理
  - わからない（つかみどころがない）
- タイプのたとえ（どちら寄りか）:
  - 猫っぽい / 犬っぽい
  - 草食系 / 肉食系
  - 太陽っぽい / 月っぽい
  - 火っぽい / 水っぽい
  - 主役タイプ / 脇役タイプ
  - 王道 / 個性派
  - 大人っぽい / 子供っぽい
  - 男性的 / 女性的（中性的）
- たとえ（型でつかむ）:
  - 動物にたとえると（猫・犬・うさぎ・狼・熊・狐・ライオン・亀・馬・鳥 など）
  - 四気質: 多血質（社交的・楽天的） / 胆汁質（情熱的・短気） / 憂鬱質（繊細・完璧主義） / 粘液質（冷静・マイペース）

## 外見・容姿

- 特徴（観察）: 顔立ち / 体型 / 髪型 / 服装 / 肌・血色 / 化粧 / 匂い ← 細目は 人を見る要素.md
- 印象:
  - 美しい・かわいい・かっこいい / 見た目がいまいち
  - 清潔感がある / だらしない
  - 若い / 大人っぽい
  - 健康的 / 不健康
  - 体が鍛えられている / 弱々しい
  - 派手 / 地味
  - フォーマル / カジュアル
  - 都会的 / 素朴
  - 色気がある / 枯れている（セクシー / 清楚）
  - 良い匂い / 気になる匂い

## 雰囲気・佇まい（表情・姿勢・所作）

- 特徴（観察）: 表情 / 姿勢 / 所作・しぐさ / リアクションの大きさ / 目線
- 印象:
  - 穏やか・落ち着いている / そわそわ・落ち着きがない
  - 余裕がある・悠々 / 余裕がない
  - りりしい・凛としている / だらしない
  - しなやか / こわばっている
  - 隙がある / 隙がない
  - 動 / 静
  - エネルギッシュ / 物静か（テンションが高い / 低い）
  - 明るい / 暗い
  - さわやか / 暑苦しい
  - ポジティブオーラ / ネガティブオーラ
  - 感情表現が豊か / 乏しい（表情豊か / ポーカーフェイス）
  - 目が生きている / 目が死んでいる
  - お茶目 / 真面目

## 話し方・コミュニケーション

- 特徴（観察）: 声の高さ・大きさ / 話す速さ / 間 / 口癖 / よく話す話題 / 言葉遣い
- 印象:
  - 話しやすい / 話しにくい
  - 声が良い / 声が気になる
  - 面白い・話が面白い / つまらない
  - ノリがいい / 悪い
  - ボケ / ツッコミ
  - 空気を読める / 読めない
  - レスポンスが早い / 遅い
  - 言いたいことが言える / 言えない
  - エンタメ・ムードメイカー / 聞き役・大人しい
  - コミュ力がある / 人見知り
  - 陽キャ / 陰キャ

## 性格・内面

- 特徴（観察）: ふだんの態度 / 何に怒る・喜ぶか / こだわり / 態度の一貫性
- 印象（人柄）:
  - 優しい / 厳しい
  - 愛がある・情がある / 無愛想・ドライ
  - 穏やか / 気難しい
  - 精神的に安定 / 不安定
  - 社交的 / 内向的
  - お人好し・人がいい / 計算高い
  - さっぱり・サバサバ / ねちっこい
  - 裏表がない・天然 / 裏表がある・計算
  - 誠実 / 不誠実（正直 / 嘘つき）
  - フェア / ずるい
  - 義理堅い / 薄情
  - 常識がある / 非常識
  - 開いている（人懐っこい） / 閉じている（壁がある）
  - 社会性がある / 社会性が乏しい
  - まとも / 歪んでいる・壊れている
  - 潔い / 見苦しい
  - S / M、攻め / 受け
- 印象（行動傾向）:
  - 芯がある・頑固 / 流されやすい
  - 自信がある / 自信がない
  - 野心がある / 無欲
  - 自立している / 依存的
  - リーダータイプ / フォロワータイプ
  - 論理的 / 感情的
  - 直感的 / 分析的
  - 楽観的・ポジティブ / 悲観的・ネガティブ
  - 慎重 / 大胆
  - 臆病 / 勇敢
  - 几帳面 / おおざっぱ
  - せっかち / のんびり
  - 世話焼き・気が利く / マイペース
- 性格の良さからくる印象:
  - 頼もしい・包容力がある
  - 安心感がある
  - 癒される・愛が深い
  - 甘え上手
  - 応援したくなる・頑張り屋・努力家

## 能力・スキル

- 特徴（観察）: 何ができるか / 実績 / 仕事ぶり / 詳しい分野
- 印象:
  - 賢い・頭がいい / 頭が悪い
  - しっかりしている / ぼんやりしている
  - 仕事ができる・器用 / 不器用
  - 多才・引き出しが多い・博識 / 一芸
  - 瞬発力がある / じっくり型
  - センスがいい / 悪い
  - 強そう / 速そう / 機能的
  - 専門特化（スペシャリスト） / 何でも屋（ジェネラリスト）
  - 理論派 / 実践派
  - 努力型 / 天才肌
  - 視野が広い / 狭い
  - 思考が深い / 浅い
  - 機転が利く / 鈍い
  - 〜が上手 / 下手
  - 〜の才能がある / ない

## 価値観・関心

- 特徴（観察）: 好きなもの・嫌いなもの / 興味の対象 / ハマっていること / 大事にしていること
- 印象:
  - 〜が好き / 嫌い
  - 〜に興味がある / ない
  - 〜が良い / 悪いと思っている
  - 刺激を求める / 安定を求める
  - 好奇心旺盛 / 無関心
  - 現実的 / 浮世離れ
  - 保守的 / 革新的
  - 堅実 / 挑戦的
  - 個人主義 / 集団主義
  - 物事を俯瞰している（一歩引いている） / のめり込む

## 地位・ステータス

- 特徴（観察）: 肩書き・役職 / 所属（会社・団体・コミュニティ）/ 持ち物（時計・車・ブランド）/ 立ち居振る舞い
- 印象:
  - 地位が高そう / 地位が低そう
  - 裕福そう / 質素
  - 育ちが良さそう / 苦労してそう
  - 品がある・気品がある / 下品
  - 影響力がある / 影響力がない
  - 一目置かれている / 軽んじられている

## 自分との関係

- 特徴（観察）: よく会う / あまり会わない、顔が広い、どんなやりとりがあるか
- 印象（関係の力学）:
  - 上 / 下に見ている
  - ギバー / テイカー（マッチャー）
  - 恩がある・用がある
- 印象（やりとり・実績）:
  - 誘えば来る / 誘っても来ない、誘われる / 誘われない
  - 助けてくれる / 助けてくれない
  - 褒めてくれる / 褒めてくれない
  - 一緒にいると楽 / しんどい・息苦しい
- 印象（自分にとっての位置づけ）:
  - 応援したい / 守りたい
  - 抱ける / 抱けない

## 今の状態（一時的）

持続的な性質ではなく、その時々で変わる印象。

- 機嫌・気分: はしゃいでいる / 落ち込んでいる・元気がない / 緊張している / 忙しそう / 機嫌が悪い
- 体調: 肌にツヤがある / 疲れている / 声が枯れている / しんどそう

## 記録するなら（より具体的に）

特定の人について書き留めるときの枠。

- この人といえば◯◯（口癖・仕草・見た目など、思い出すフック）
- 出会いの文脈（いつ・どこで・どう知り合ったか）
- 誰の知り合いか・どこの所属か
- 印象に残っているエピソード
- どんな人か
- 良いところ・好きなところ
- 悪いところ・嫌いなところ（直してほしいところ）
- 困っていること
- しそうなこと
