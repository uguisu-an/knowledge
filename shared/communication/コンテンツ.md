# 内容 Content

チャネルやメディアを通じて伝達される内容。メッセージ。

コミュニケーションで重要なのは内容そのものではなく、相手の意図。相手がそれを伝えることで何をしたいか——世界を良くしたい、相手を貶めたい、騙して利益を得たいなど、様々な意図がある。

曖昧な内容を選ぶなら、相手がそれを読み取れるかどうかが重要。読み取れないならもっと明確な内容で伝えないといけない。

感覚を知識として伝えたらむしろ理解の邪魔になることがある。

## 種類

- Conversation: 言葉を使う
- Promotion: なんらかの行動を促進・助長する目的で行う
- Narration: 物語やエピソードを伝える
- Discussion: 各々の持論を比較する
- Education: 文化を継承する
- Operation: 操作する
- Presentation: 形式的なコミュニケーション

## 書かなくていいことを判別する

ライティングにおいて、書かなくていいことを判別することは重要なスキル。

- テーマとメインポイントは明確か
- 内容が統一されているか。不要な詳細や重複した情報はないか
- 文章が冗長ではないか。不要な修飾語やフィラー語は削除できるか
- 読者のニーズを考慮しているか
- クリアで理解しやすい言葉を使っているか
- 根拠やサポートが適切か
- 文章の構造が整っているか。論理的な流れがあるか
- 言葉遣いや文法に間違いはないか
- 文章が興味深いか。引用、エピソード、例が含まれているか
- 要約すべきか
