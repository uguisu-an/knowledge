# Context

文脈によってメッセージの意味も異なる。

問題や文脈を切り分ける。

うさぎ「なんだ？文脈読めないのか？話聞くの上手じゃないんだな。人間なのにな。」（『＋チック姉さん』）

- 目的: 何をしたいのか
- 背景
  - 意図: なぜそれをしたいのか
  - 経緯: なぜそれをしてきたのか

職場で「文脈が読める人」になる方法
https://x.com/ysk_motoyama/status/2024978444711592070

ムードも文脈の一種。

コミュニケーションの核心は文脈にある。
目的と背景。

何をしたか、
何に困っているか、
などの文脈、生活形式を共有する。
相手が「お腹が空いている」から食事の話題になる。
相手の中に文脈はすでにある。
文脈というのは、相手の関心。
