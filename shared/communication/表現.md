# 表現 Expression

形式化された表現もあれば、カジュアルな表現もある。

コミュニケーションのための表現は一般的なものというよりも presentation の範囲。
expression もその一部ではある。

表現というよりは提示方法というべき？

description は視覚的なイメージを想起させる表現。

良い、悪い、というのは抽象的すぎるので、どう良いのか、どう悪いのか、まで言葉にする。

内容がアレだと表現ではどうにもならないこともある。

表現が変わっても、そもそもの思想が変わってないこともある。

表現も根っこは何らかの影響を与えること。コミュニケーション。

「人はニュース性に飛びつきます。同じことを言う場合でも、言い方を少し変えるだけで、途端にニュース性を帯びてきます。」
https://toyokeizai.net/articles/-/817148?display=b

- 時事性: 生じて間もない
- 公共性: 多くの人に関わる
- 人間性: ヒューマンドラマがある
- 意外性: 驚くような要素がある
- 希少性: 珍しい
- 緊急性: 時間制限がある

自分に向けた表現が他人に全く届かないってことはないと思う。ただ、届きづらい可能性が高い。

どう解釈したか、どんな体験を想起したか、どう表現したか。表現の基礎はここ。
