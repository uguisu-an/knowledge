# Determiner

対象が限定されているかどうか、定性 (Definiteness) を表す要素。

以下の語が含まれる。

- 冠詞（the）
- 指示限定詞（この、あの、その）
- 所有限定詞（私の、あなたの）
- 疑問限定詞（どの、どんな））
- 否定限定詞（no）
- 数量詞（X個の、X人の）

英語的には、定である対象には定冠詞theをつける。
日本語的には、不定である対象には助詞「〜は」を使えない。

定であるということは、旧情報、既存の情報であると考えることもできる。

数量詞が限定詞として扱われる例としては、街中で「そこにいる4人の男」のように使って対象を絞れるようにするとか。視界に入っている人の中で4人で固まっている男だけを絞り込める。

日本語は冠詞や否定限定詞を持っていない。
