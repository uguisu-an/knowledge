# 比喩 Metaphor

直接的に伝えることができないことを、別の物事やイメージを使って表現する。
2 つの異なる事物を比較することにより、新たなイメージやイディオムを作り出す。

隠喩と違って、明確に二つの事柄を比較する。

比喩は修辞法の一種。
直喩、反直喩、暗喩、などの種類がある。誇張法や擬人法も比喩の一種。

比喩も感じ方なのかな。でも表現かも。
比喩は表現ではあるけど、感じたことから比喩を展開させるのでそもそも感じ方も重要。

比喩は、抽象的な印象を具体的な表現にしてくれる。

ころさんがぐるぐる絡まったホースを見て「川崎ジャンクションやん」って言ってたのは面白かった。ホースの形をJCTに例えるだけでなく、「川崎」をつけてさらに具体的にしているのもポイントが高い。

比喩を兼ねた描写。カビの臭いや暗い路地などは陰気さを演出する。事実を描写しながらどんな印象に注目すればいいかわかりやすい。
