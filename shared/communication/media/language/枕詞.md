# 枕詞

ある特定の言葉を導くために添えられる決まった言葉。

枕詞の種類:

- きっかけを作る
- 予防線を張る
- 切り込む
- 踏み込む
- 思考を促す

「厳しいことを言うようだけど」
「アドバイスとして聞いてほしいんだけど」
「オチがないんだけど」
みたいな枕詞で相手に覚悟や拒絶の時間を与えるのも気遣い。最初に結論を言うのと近い。話のオチを先に伝えている。

「XXXの話に戻るんだけど」

「これ言っていいかわからないんだけど」

和歌などの修辞技法では通常5音節。「ひさかたの（光）」や「ちはやふる（神）」などがある。

「ぬばたまの」は美しく黒いものにかかる枕詞。
「ぬばたまの黒髪山に〜」と言ったとき、「黒髪山」自体は固有名詞だけど、「ぬばたまの」が「黒髪」にかかることで「黒髪」を強調する。景色としての「黒髪山」と人の特徴としての「黒髪」を同時に語る。

話が長くなるなら前置きする。
「説明するのは難しいけど」とか「少し長くなるけど」とか。

「嫌味なこと言っていい？」みたいな枕詞を言うべきかどうか。そもそも嫌味なことを言うべきかどうか。
