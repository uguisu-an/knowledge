# 英語の発音

英語の音節は基本的に 子音-母音-子音 をワンセットで扱う。子音-母音の日本語とは違う。
例えば、information であれば、 in.for.ma.tion. で切れる。
strike なんかは 1 音節しかない。

日本語のように、一文字が一音節の言語を基準にすると違いが大きい。

発音と文字の対応ルールとして、「フォニックス」と呼ばれるものがある。
だいたいこの通りに発音しておけば問題ない。

聞き取りで問題になるのは単語の連結。
"take out" が「テカウト」のように混ざる。
連結のない日本人にはとてもつらい。

英語の発音は連結と脱落があるのでみたままの音にはならないことがある。

iの発音は「い」の口で「え」の舌。

英語の発声はまず腹式呼吸から。

英語を話すときは、顎を動かさない。舌で発音をコントロールする。日本語はどちらかというと顎と喉を動かす。

日本語訛りの英語はそもそも息が足りないことが多い。腹式呼吸でしっかり声帯を鳴らすところから始める。

英語は強調したい母音をどれだけ伸ばしてもいい。
「アーーイハバドリム」でもいいし、
「アイハブエーーイドリム」でもいいし。

ngは鼻濁音。鼻の奥で「ん」を出す感覚が強い。

大事な単語が長く読まれる。

英語は息を止めない。発声してないところでも息を吐いている。

英語はtheとかaとかthisとかどうでもいい単語が増えても音の数が変わらない。重要なところにしかアクセントがない。

母音の文字が明らかに足りないことからもわかるように、英語やアラビア語は必ず音が先で字が後。音から覚えないといけない。

英語はアクセントのある母音が伸びやすい。
「ファンタースティック」とか「アーブソルートリ」とか。

日本語と英語は母音の差が大きい。
あと英語は母音が20種くらいあるのに表記が5種しかない。表記を見ても発音を知らないと発音できない。

英語は単語や熟語のアクセントが重要なので、そのためにも呼気圧をしっかり腹式呼吸で出す。アクセントは基本的に破裂音になるので、声帯で受け止めた息を解放する。

英語のKは日本語のか行よりもっと奥で鳴らす。というか腹式呼吸で鳴らす。

英語の発音はまず喉を下げて、顎を動かさずに舌で発音を変えるのが基本。

Oの発音は日本語なら「ァォ」に近い。

英語には母音が15個ある。普通にアルファベットが足りてないので、アルファベットを学ぶより前に発音を学ぶべき。
