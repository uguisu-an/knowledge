# Value In Exchange

- 価値を提供する
- 価値を回収する

交換できるものには価値が生じる。

交換によって、豊かさを得られる。

交換が成立するためには、価値観の相違が役に立つ。

何者かになるためには、相手がいないといけない。
誰しも「誰かにとっての何者か」にしかなれない。

経済の基礎はexchange、価値の交換。提供と回収。ビジネスもそれを利用することが多い。
ビジネス自体は、実は経済を基盤にしなくてもいい。寄付とか、親や配偶者に養われるとか。

- 誰が一番高く買うか。
- 誰が欲しがっているか。
- 何が欲しいか。
- 自分は何を持っているか。
- 自分は何を与えられるか。
- 相手は何を持っているか。
- 相手は何を求めているか。

価値を交換するには相手が必要。

恋愛や商売は価値の交換がよく現れる。

誰彼構わず価値を提供するのが交換の第一段階。とにかく押し付ける。できれば相手の欲しいものがいいけど、必須ではない。

交換が成立するのは、こっちが価値を提供して、相手から価値を回収したとき。相手が価値を提供しようとするか、契約で縛るか。

交換の基礎はまず与えること。信頼。期待感と安心感があること。
その上で価値を回収できる相手に絞っていく。

交換の順序:

1. 価値を提供する
2. 価値を回収できない相手を切る

価値を提供するばかりで回収できないと交換にならないし、価値を提供せずに回収することばかり考えていてもうまくいかない。
まずは相手に限らずとにかく価値を提供して、回収できなかったら切っていく。与える前に選ぶのではなく、与えた後に選ぶ。戦略的ギバー。

仲介者が交換を仲介する場合がある。

信頼は交換の前提条件であり、交換の産物でもある。
交換によって信頼を得るには、回収できるかどうかわからなくてもまず提供から始めるといい。

交換によって信頼が複利的に積み重なって大きな取引を可能にする。

交換の種類:

- 双方的・一方的
- 提供と回収の時間差
- 仲介者の有無

信頼自体は交換以外の文脈でも使われる。

交換があったら価値の交換もありうる。でも必須ではない。
