# Value In Exchange

- 価値を提供する
- 価値を回収する

交換できるものには価値が生じる。

交換によって、豊かさを得られる。

交換が成立するためには、価値観の相違が役に立つ。

何者かになるためには、相手がいないといけない。
誰しも「誰かにとっての何者か」にしかなれない。

経済の基礎はexchange、価値の交換。提供と回収。ビジネスもそれを利用することが多い。
ビジネス自体は、実は経済を基盤にしなくてもいい。寄付とか、親や配偶者に養われるとか。

- 誰が一番高く買うか。
- 誰が欲しがっているか。
- 何が欲しいか。
- 自分は何を持っているか。
- 自分は何を与えられるか。
- 相手は何を持っているか。
- 相手は何を求めているか。
