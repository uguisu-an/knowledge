# Vibe

勢い。
活性。

ノリは個人の問題で、空気は全体の問題。

ノリが悪いと空気が悪くなりやすい。

感情的なことは勢いで進めた方がうまくいくこともある。恋愛とか。

勢いを重視すると、視野が狭くなりやすい。

勢いはworkかも。

勢いは働きの文脈でもいいっちゃいい。

勢いがある時は勢いを止めない方が力をうまく使える。ただ、力が強くても向きが悪いとむしろ悪い方に向かってしまうこともあるし、止まりきれずに崖から落ちることもある。

ノリよりもムードを重視する。

ノリに関しては音楽と関連する部分がある。テンポが速くて抑揚が大きいほうがノリが良い。

会話のノリを維持するために速いテンポは必要かどうか。ゆっくりしたテンポでもいいかどうか。

ノリは別にどうでもよくて、ノリが悪い人に対して態度を悪くしてムードを悪くするのはクズ。

ノリには重さ、軽さがある。ノリが軽いほど勢いに乗りやすい。普通はそこまで軽くない人が多いので、助走が必要。

ノリがいい、と言われるのは、気さくさ、楽しさ。一緒にいて楽しいこと。

ユーモアが大事。
まず、楽しませようとすることから始める。滑っても、恥ずかしくても、慣れないといけない。

つまらない話を聞いてつまらない人が笑ってたらそれなりに盛り上がってる感じになる。

足手まといがいるとノリが悪くなって盛り上がりにくい。
いっそのこと会話から排除するのも一つの手ではある。

ノリが高くても、ムードが悪いと空騒ぎになる。
ノリよりもまずムードをよくする。

動きたい気分、活性がノリ。勢い。

狂気はvibesの一部。

ノリはその人の声とか体力とかもあるので、高ければ良いというわけではない。無理にノリを合わせようとしても長続きしない。

ノリは良ければいいわけじゃない。ノリが合わないと付き合いづらい。

どういう行為についてノリが良いのか、によって合う合わないがある。
人をいじるのが好きな人とノリが合わないとか、よくある。褒め合うノリが気を遣ってしまって合わないってこともある。
