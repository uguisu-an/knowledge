# 目的 Ends

ある行動を「なぜするのか」を示すもの。

どのように変えたいか。理想の状態。

最終的に獲得したい状態や成果。

理念やビジョンとして設定される。

できないことや困っていること、やりたいことや好きなことから目的を決める。

どちらにしても将来を考えるところから始める。

「なんのために」を積み重ねることで上位の目的を探る。

目的の達成というのは、理想の実現でもある。

人間でも機械でもまずは目的の設定が重要。目的が違えば手段も変わる。

目的を決める第一歩はその日の体調、予定、気分でいい。

目的というのは、達成できなくても近づくことに意味がある。
問題解決、理想の実現、に近い。

どちらかというと手段が先、目的が後。
目的がない場合もある。あるいは続けるために続けている、とでもいうべきか。

大きく分けて、続けるために続けていることと、終わらせるために続けていることがある。
achievement と entertainment の関係。

目的を遡ったときにそれ以上目的がなかったら無理にこじつける必要はないのかも。

目的のない仕事もある。適当に変えて遊ぶ。

achievement にも目的がないといけない。

手段に対して「なぜそうするのか」を考える人、目的を考える人は本当に目的を達成しようとしている。
そうでなければ、目的と手段が逆転してしまっている。

何を変えたいか、理想や問題が目的になる。
あるいは変えないことも目的になる。

目的を設定するのは decision の文脈。

目的思考は利益思考なのかも。

目的がなければ行動とはいえない。

目的を重視し過ぎるとコミュニケーションにおいて必要以上に情報の削ぎ落としが起きてしまう。

行動の文脈としては、目的があれば目標はあってもなくてもいい。
目標は achievement の文脈なので、何らかの実績を残したいならあってもいい。
ただ、理想の状態のために成果が必要なこともある。

目的とは、理想の状態のこと。

人が求めているものは様々。
とくに豊かな国だと多様化している。
多様性を理解していないと、目的が揃わないまま始めて混乱する。

人生に価値や意味がないとしても、何らかの目的を仮定しておくと仕事とかコミュニケーションとかに便利。

目的は達成するものというよりも、近づくためにある。

やりたいこと。

「何のため」を考えるのは action や decision.
目的と手段、判断と行動、をまとめるところがあってもいいかも。

価値が行動を決めるところはある。
普通は良いと思った方を選ぶ。

いきなり「どうしよう」と考えているときは、まだ目的が見つかっていない。
まずは目的を明確にしてから、「どうやって」を考える。

work, game, loveにはそれぞれ目的がある。
変化、勝利、満足、かも。
