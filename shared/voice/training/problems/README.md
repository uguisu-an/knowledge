# 声の課題

問題がある声の出し方は避ける。

まず、声が枯れる、喉を痛める出し方は絶対に良くない。

- 声が震える、安定しない
- 声が裏返る、高音で声が細くなりすぎる
- 口が痺れる
- 声が枯れる
- 息が漏れる
- 息がきれる
- 母音が潰れる、発声が悪い
- 高音で声が強くなりすぎる、余計な力が入る
- 発音によって音程がズレる、響きが変わる
- 暗い声しか出ない
- 明るい声しか出ない
- 母音を変えると音色が変わる
- 音程を変えると音色が変わる
- 音程を変えられない
- 音量を変えると音色が変わる
- 音量を変えられない

声の色んな属性を変えるために、一つだけ変える練習をする。音色や音程を変えずに音量だけ変えるとか。

標準的な声の出し方でなくても声が良ければ歌手や声優として大成することはあるけど、真似するべきではない。
真似をするなら伝統的で安全な声の出し方を真似する。

優里や虹色侍みたいにずっとがなって無理やり高音を出すような歌い方はどうせ5年10年しか寿命がないので真似してはいけない。
