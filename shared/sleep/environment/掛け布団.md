# 布団

タオルケットで羽毛布団や毛布より一段階涼しい構成も取れるようになる。

五段階で調整する。

1. タオルケット
2. 毛布
3. タオルケット＋毛布
4. タオルケット＋羽毛布団
5. 毛布＋羽毛布団

梅雨の時期は布団の調整が難しいので、除湿で寒いくらいに調整して、布団は暖かめに揃えた方が楽か。

夏場は薄手の掛け布団があると便利。
冷房を強めにかけるなら毛布でもいい。

布団の組み合わせ案:

1. 毛布/羽毛布団/毛布/人/毛布
2. 毛布/羽毛布団/人
3. 羽毛布団/人/毛布
4. 羽毛布団/毛布/人
5. 羽毛布団/人
6. 毛布/人

毛布は変にヒートテックみたいな発熱素材にすると暑すぎて寝苦しくなりがち。

毛布は綿素材が夏でも腹かけに使えるくらいで、カシミアやウールだともっと暖かい。
部屋の温度、湿度、使用者の好み、汗をかくかどうかとか体温なんかも関係するので、安いのを色々試してみるといいかも。
