# Fun

ポジティブな体験や、それを引き起こす要因。

誰かを楽しませて、それを共感できれば、能動的に楽しめる。

楽しいのは肯定的なフィードバックがあるとき。行動が強化されたとき。

誰かを楽しませるには、意識させずに学ばせる。そこで重要なのはフィードバックやリアクションだ。反応がないのは一番つまらない。

楽しむには学ばないといけない。刺激を求める。

相手を楽しませるのは、退屈でなくするということ。

楽しみはひとつじゃない。多ければ多いほどいい。

楽しまないと生きられない。人間にとってはもはや楽しむことが義務とも言える。

状況を楽しめていないということは、逃げ腰になっているということ。

「もしひとつの「楽しい」が終わらなかったら、次の「楽しい」を探さなかったと思う。」『瑠璃の宝石』

楽しむことは、続けること。続けることは、楽しむこと。

時間が過ぎるのが早いと、楽しさを感じている。

「楽しいから笑うのではない。笑うから楽しいのだ。」
『ウィリアム・ジェームズ』

楽しかったら、「足りない」「もっとやりたい」と思う。
