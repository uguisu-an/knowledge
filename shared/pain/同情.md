# Sympathy

他人の状況を思いやり、支援する能力。

empathy や sympathy は共感と訳されることもあるけど、empathy は respect の方が近い。

同情は共感というよりもCareの心。世話しよう、大事にしようと思うこと。

「同情するのは無料だ。」『水属性の魔法使い』

かわいそうに見られたくないなら、かわいそうに振る舞うべきではない。
かわいそうに振る舞っているなら、本心ではかわいそうに見られたいということだ。それならかわいそうに見られることを素直に喜べばいい。

共感は自分ごととして考える。
同情は他人ごととして考える。

「かわいそう」みたいなのも他人事。「かわいそうやね、知らんけど」と言っているのと同じ。
