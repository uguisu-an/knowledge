# Help

助け。

誰かの痛みや苦しみを和らげる (ease) と救いになる。

好ましくない状態を脱すること。

危険な状態を脱すること。

恩 favor, 同情 sympathy, 痛み pain あたりは近いかも。

弱い人は助けられて当然だと思っているので、助けてもメリットがない。放っておいた方がいい。

弱者は助けたくなるような姿をしていない。

全てを救わなくていい。

ヒーローはいつも遅れてやってくる。

医療だけでなく、救助の段階でも助けられる人を選別する。リソースは有限である。

人を助けるのはその人も資源として見れるからであって、人を資源として見ない人が誰かを助けているのはおかしい。何か裏があると思っていい。

甘さは弱みでもあるが、誰かにとっては救いでもある。
