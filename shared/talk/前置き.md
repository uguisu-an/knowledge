# 前置き

「厳しいことを言うようだけど」
「アドバイスとして聞いてほしいんだけど」
「オチがないんだけど」
みたいな枕詞で相手に覚悟や拒絶の時間を与えるのも気遣い。最初に結論を言うのと近い。話のオチを先に伝えている。

「XXXの話に戻るんだけど」

「これ言っていいかわからないんだけど」

話が長くなるなら前置きする。
「説明するのは難しいけど」とか「少し長くなるけど」とか。

「嫌味なこと言っていい？」みたいな枕詞を言うべきかどうか。そもそも嫌味なことを言うべきかどうか。

相手にやってほしいことを明確に限定する意味もある。

- 結論から先に言います。
- 前提だけ共有します。
- 判断は任せます。
- 確認だけお願いします。

枕詞で「どういう反応をしてほしいか」を伝えることもある。例えば「意外かもしれないけど」だったら、驚いてほしい。

- 説明すると長くなるんだけど
- 意外かもしれないけど
