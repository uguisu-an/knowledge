# Talk

話、噺、喋り、語り。

一般的な「喋る」「語る」がtalkにあたる。

言葉を使ったコミュニケーション。
双方向とは限らない。双方向はconversation.

話よりも歌が先にあると思う。
歌からメロディやハーモニーを減らして単純化したのが話。

英語の talk は聞き手がいて、speaking を含む。
speaking は言葉を発すること。聞き手がいなくてもいい。

一方的に話すだけでも talk になる。

talk は聞き手が言葉を発さなくても talk だけど、conversation は言葉のやり取りで、お互いに言葉を発する。
だから、漫談みたいな一方的に話すものでも talk と呼ぶ。
逆に、雑談みたいに何かを伝える目的でないものは talk というよりも conversation に近い。
自分が話して、相手に何かを伝えるが talk と考えるのがいいかも。

語る、のほうが近いかも。

トーク力はそもそも観察の解像度の時点でおおよそ決まっている。面白いネタが全て。あとは簡単な技術があればいい。

トークが面白いのと、その人の感覚自体が面白いのとはまた違う。

speaking は会話だけじゃなくて、言葉にならない発声や生体模写なども含まれる。

身振り手振りも含まれる。

相手の話を聞くような質問も talk の一部。
conversation になっていく。まず talk が成立してから、conversation になる。

1:1で話すのは特にdialogと呼ぶ.

信頼を目的としているのか、娯楽を目的としているのかでも違う。
信頼を目的とするなら一人一人に語りかけるのが大事。政治的なスピーチとか。
エンターテイメントの場では喋っているのを全体に見せる。漫才とか舞台みたいに。

しゃべりには音楽的な要素がある。
しゃべるときも音楽と同じようにリズムやメロディを意識して、一定のテンポを取りつつ、リズムを変えたり、アクセントをつけたりする。

ペースが速すぎると聞きづらく、遅すぎると注意を引きづらい。

相手の目を見て、大きな声で喋る方が信用される。

エピソードが面白かったらトークがほどほどでもただ順序通りに話すだけで面白い。

普段から、なぜそれを面白く感じたのか、なぜそれを良く思ったのか、を考えておく。

印象だけでなく、具体的な事象として、何があったか、いつ、誰と、どこであったか、も詳しく覚えておくと話題にしやすい。

印象や感想は、事象をまとめてしまっているので、話を広げるならまとめる前の生の状態の方が広げやすい。

相手に合わせるばかりではつまらない。まずテンポを合わせて、慣らしてからゆっくりと自分のテンポに持ち込む。

積極的に話ができなくても会話はできる。
少なくとも聞かれたことに答えられればいい。話題が途切れないようにする。

どんな振る舞いが話し上手か。
トークが上手なのと、会話が上手なのは違う。

talkとconversationは区別する。
talkができなくてもconversationはできる。talkがつまらなくても楽しいconversationはできる。

語れば語るほど感情も知識も豊かになっていく。グラデーションが細かくなって、名前がつくことで区別されていく。

何かを語るには、ながら作業しない方がいい。

語ることはまた考えることとは別のスキルが必要な気もする。

冬に間違って屋根のないカフェで集合してしまって、
「風強すぎる」
「クソさみぃ」
というのはfeelingだけど、
「クリスマスツリーがブンブン頭振ってる」
「誰もいねえ」
みたいなのは出来事。でも出来事も補足した方が解像度を上げられる。
これは出来事自体が面白かったパターンの話し方。

喋りと声楽は似たところがある。

むしろ声楽は朗唱から派生したものと考えられる。
韻を踏んだりするのは歌唱というよりも朗唱の特徴の一つ。

まずは内容よりも喋り方が大事。
発声であったり、間であったり、言葉遣いであったり。

考えながら喋ると相手から目を離しやすいので、まず考えてから、相手を見て話すといい。
ただ、短期記憶が弱いと考えたことを持っておけなくてこれができない場合もある。

滑舌が悪かったらゆっくり喋るといい。

SpeakingはSingingの派生。言葉を喋るなら、どっちにしてもリズムが必要。ただ、メロディやハーモニーをそこまで重視しない。

話が上手い人は歌も上手い人が多い。リズムや抑揚の取り方が共通している。ただ、音感が悪すぎるとその限りではない。

人の話を聞く能力が低い人に囲まれて過ごすと、とにかく早口で喋る癖がつきやすい。

何を目指すかによっても話し方は違う。

話はspeakingとは限らない。文字などで話すのもtalkだ。

手振りと頷きでリズムをとりながら喋るのはいいかも。

言葉が足りないと聞き手にとっては補完しないといけないことが多くて大変。

話としては、言語化とか抽象化とか具体化というよりは、言葉の量を増やしたり減らしたりできるのが大事。言葉少なに表現することもできるし、長い言葉で表現することもできる。

物語も話。トーク。
