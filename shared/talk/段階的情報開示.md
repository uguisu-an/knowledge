# Progressive Disclosure

1. キャッチコピー: 一言
2. エレベーターピッチ: 15-30秒
3. スモールトーク: 1-3分

段階的情報開示。

まずは「え？」っと思わせるようなキャッチコピーから入る。
この段階で興味を持てない話はそもそもつまらない。

次にエレベーターピッチやで面白みを伝える。その話の何が面白いのか、を伝えるのが大事。
ここでさらに「もっと聞きたい」と思わせるのが大事。
相手のリアクションが良くなったら、スモールトークでもう少し詳しく話す。
ここまでくると相手から質問してくるので、あとは質問に答えるだけで十分。

エレベーターピッチは、あらすじではなくて、「何が面白いのか」を伝えることに特化する。
面白みは分野によって違う。ビジネスなんかでも面白みというのは決まっていて、「金になりそう」な話が面白い。

行動や出来事自体が突飛だと「こういうことがあった」ってただ伝えるだけで「え？」ってなる。
そうでないならキャッチコピーを考えないといけない。

相手から質問させる。
話したあとは相手が質問できるように間を取る。

相手に「もっと聞きたい」と思わせてから、次の段階に進む。
相手の反応で判断する。
