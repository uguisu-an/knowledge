# GREAT Method

- Goal: 聞き手にしてもらいたい行動
- Reason: 話し手の目的
- Example: 具体的な行動、制約条件
- After That: その行動によって聞き手にどんな得があるか

例えば、「この資料を10部コピーしてほしい。（Goal）14時の会議でクライアントに配るから（Reason）、両面印刷のカラーで、クリップで留めて一部ずつファイルに分けて。（Example）お礼に晩飯奢るから。（After That）」みたいな。
