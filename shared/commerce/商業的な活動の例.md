# 商業的な活動の例

## 価値を作る

- 市場調査
- 製品開発
- 調達
- 製造
- 品質管理
- ブランドや知財の構築

## 価値を伝える

- マーケティング
- ブランディング
- 広告・宣伝
- 宣伝用のコンテンツ作成
- SEO
- SNS
- 広報活動
- リードジェネレーション

## 需要を見つける

- 販路開拓
- 販路設計
- 流通・卸売
- 仲介
- マーケットプレイス運営
- 提携
- 輸出
- 輸入

## 取引を履行する

- セールス
- 交渉
- 価格設定
- 見積もり作成
- 契約
- フルフィルメント
  - 受注管理
  - 在庫管理
  - 注文された商品の探索
  - 梱包
  - 発送
- 配送・物流
- 決済

## 関係を続ける・広げる

- カスタマーサポート
- カスタマーサクセス
- 解約防止
- アップセル
- クロスセル
- ロイヤルティ・コミュニティ運営
- 口コミ促進

## その他の支援活動

- 財務・会計
- 資金調達
- 収益モデル設計
- データ分析
- 法務
- コンプライアンス
- 与信管理
- 人材・組織
