# 評価

価値を決めること。

価値は需要と供給から決まるので、主に需要を想定する。

個人の需要と市場の需要は異なるので、自分は価値がないと思っても一般的には価値があることもある。

第三者からの評価を公開するなら、評価自体の信頼性を求めるサクラチェッカーみたいな仕組みがないと意味がなくなる。

社員や商品が良い評価をされないのは特別に評価される要素がないからで、悪い要素があるわけではないこともある。

評価するから価値があるのではない。

check

評価は点数というよりも値段でつけるべき。自分なら、あるいは市場ならいくら出せるか、をつける。

点数で評価する場合、それなりに良かったら全部100点でいい。それ以上は比較する必要もない。

本人にとって、後世になってから評価されるのってどうなんだろう。
まぁ、子孫や親類には財産を残せるのかも。

安易に評価をくだす人の評価は安い評価だ。重要ではない。

- 誰が評価、判断するのか
- どうやって評価、判断するのか

他人を評価する人間は恨まれても仕方ない。

評価は判断。順番をつける。

俺を評価するのはまず俺である。

定量評価だけでは足りないけど、定性評価は難しい。特に組織では主観だけで決めてしまうと良くない。ちゃんとやろうとすると、評価者を増やして平均化するとか、手間がかかる方法になる。

他人の評価よりも自分の評価が低くあるべき。

自分の評価と他人の評価はズレることがある。

「評価」にあたる英単語の種類:

- evaluation: 数字をつける
- rate: 連続的な数値評価を決める、点数をつける
- rank: 段階的な階級を決める
- review: 見直し
- assessment: 詳細に分析して良し悪しを分ける

evaluateが一番広義の「評価」に近い。分析的なプロセス。
