# 強さ

強さとは、影響力の大きさ。

強さ = 美しさ x 豊かさ

使わない能力が多い → 怪しさ

使う（将来性も含めて）能力が多い → 豊かさ

使う能力が少ない → 貧しさ

使わない能力が少ない → 美しさ

どこまで将来性を含むかで、評価も変わる。

強さは複合的な要素。攻めと守り、豊かさと美しさ、などのバランスを取る必要がある。

[最初から強いやつの特徴 - Qiita](https://qiita.com/zamis/items/703bfcea027a70c1cec6)

強さを獲得するためには、適応と学習が必要。

豊かさと美しさを両立させるのは難しい。
豊かになると怪しくなりやすいし、美しくなると貧しくなりやすい。
まずは豊かさで、次に豊かさを保ったまま美しく。

何らかの分野で強くないと生きるのは難しい。

「オレは思うんだけどさあ、本当に強ぇ奴ってのは…周りまで強くなった気にさせるもんさ！」(『B-TRASH』第 2 巻)

どれだけ強くなっても足りることはない。強さを求めながら、強さに縋ってはならない。

技術的な上手さだけで強さは決まらない。

強い人は結果が安定している。
これはわざわざリスクの高い選択肢を取らなくても良い結果が出せるため。同じ選択肢でも質が高い。

強いというのは状況をコントロールできるということ。

強さには Health と Wealth、内的な強さと外的な強さがある。

強者には愛がある。
強くなるほど大きな影響を受けても崩れない。だから影響を受け止められる。

関わりを絶って余所者を排除する、影響を抑えるのは弱者の振る舞い。

強さの要素に重み、優先順位をつけるのもいい。

強さと上手さは別物。
上手・下手というのは技術の文脈。
技術があっても仕事ができるとは限らない。

プロゲーマーやプロスポーツ選手に強さは必須ではないけど、強いと注目される。

強いのと勝ちやすいのは少し違う気がする。

勝つための強さ
生き残るための強さ

食われるものの食われない戦略:

- 群れる
- 逃げる
- 隠れる
- ずらす（時期や場所をずらす）

『弱者の戦略』稲垣栄洋

勝負の世界では勝った方が強いけど、強さには勝負以外の世界もある。

強さという点で、生物には睡眠と食事という致命的な隙がある。
