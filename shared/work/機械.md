# Machine

> 機械、器械とは、広義には、ある力が有用な働きをなし、あるいは他のエネルギーの形態に変化する力の伝達を行うような装置の総称。
> 通常の用語では機械（machine）は一般に簡単な構造を有する器具（implements）または道具（tools）とは区別され、2つ以上の抵抗物を組み合わせて互いに相関的運動を行う工作物をいう。

ある力を別の力に変化させる機構が特徴。

コンピュータやその上で動くソフトウェアは機械。機械には機能と機構があって、仕事の文脈でもある。働かないと機械とは言えない。

機械式というと、歯車などの部品の組み合わせだけで、電子制御を伴わない作り。トランジスタとか使って電気信号のやり取りだけで仕掛けを作るのは電子式。
スイッチだけ電子式だけど、仕掛けの大部分は機械式ということもよくある。車とか。
機械式は部品が動くので、摩擦というか、擦り減る、摩耗するのが問題。コンピュータでもファンが機械だったりする。水冷ならほとんど摩耗する部品はない。
