# Answer

答え

既知を明らかにするもの。

全ての答えや結論は仮説でしかない。

答えを出して終わりではない。

反証が見つかるまで検証し続けないといけない。

答えを発する時は、必ず問いがあるべき。
問いでないものは常になんらかの答えだ。暗黙的な問いで、自分も意識してないことはある。

自分しか知らないことには自分で答えるしかない。

どういう前提を知れば、より良い答えを返せるか。

自分で答えを出すか、どこからか答えを得るか。

問いに答える。

同じ答えでも、練習としてはより多くの問いとセットになっていたほうがいい。
「8」が答えとしたら、「7の後は？」「9の前は？」「2の3乗は？」など。

答えを用意するだけでなく、より多くの問いと紐づけてやらないと使えない。
