# 計画作りの目的

計画を立てるのはやり直しを減らすため。

やり直しが減ってないなら計画を立てた意味がない。
計画の良し悪しはやり直しが減っているかどうかで評価されるべきだが、実際のところは計測できないので、計画を評価するのは難しい。結果的にやり直しがなかったとして、計画を立てなくてもやり直しはなかったかもしれない。逆にやり直しが多くても計画を立てる時間以上にやり直しが減っていたら計画を立てた甲斐はある。

https://qiita.com/mkt_hanada/items/3f21c850ddc02636b34c

計画を修正するのは全く問題ない。計画作りの目的はやり直しを減らすことであって、計画づくりのコストがやり直しのコストを下回っていればなんでもいい。もちろん修正が少ない方がいいけど。

計画性と柔軟性のバランス。
計画を立ててもやり直しが減らない領域というのはある。そもそも新規性が高くて予測がつかないとか、市場や法律の変化が早くて対応しないといけないとか。

計画を立てないと、目の前の小さなタスクから片付けがち。
より重要なタスクを先に処理しないといけない。
