# 儲け

純利益

儲けがあれば、貯蓄や投資を増やせる。

儲けるためには、利益・収入を大きくして、損失・支出を小さくする。

相対的に見て、「勝つ」こととも関連がある。
ただし、協力して儲けることもある。ビジネスはゼロサムゲームではない。

会計的な「利益」は margin のこと。収入から支出を引いたもの。

儲けるのは過程。
成功すること、豊かになること、実績を積んで信用を得ること、などの方が重要。

価値が高いものを作って売っているなら、労働であっても効率がいい。より少ない手間や時間で稼げる。

自分が儲けるための資本は、誰かの懐から出ている。
自分が儲けるためには、誰かを今よりも儲けさせて、能動的に余剰資金を作らないといけない。家計を大きくする。
