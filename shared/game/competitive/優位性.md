# Advantage

優位性。競争優位性。強み。有利。

- 肉体的
  - 背が高い
  - 足が速い
  - 体力がある
  - 筋力がある
  - 病気になりにくい
- 精神的
  - 恐怖を与える
  - 弱みを握っている
  - 驚かせる
  - 笑わせる
  - 壊れにくい
  - 変化に適応しやすい
    - 変更しやすい
- 経済的
  - 資本が多い
  - 市場投入が早い
  - 模倣しにくい

市場投入が早いほど競争優位性は高まる。
