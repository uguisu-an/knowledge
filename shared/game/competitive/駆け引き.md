# Tactics

戦術。駆け引き。読み合い。

相手の出方や状況に応じて、自分に有利になるように事を運ぶこと。

あえて非合理的な行動をとって騙すのも駆け引きだけど、協力ゲームの場合は仲間の邪魔になることがある。

フェイントや駆け引きはゲームの醍醐味。ゲームそのものと言ってもいいかも。押す、引く、押すと見せて引く、引くと見せて押す、など。
ゲームは対人が前提なので、読み合いがある。

情報を隠せるなら隠したほうがいい。

攻めと守りの中でもさらに駆け引きがある。

読み合いは暗黙的な交渉に近い。

人に興味がないと読み合いに弱い。

読み合いはまず太い択、合理的な択がベースで、相手が読んで裏をかくなら、自分はさらに裏の裏、というふうに読み合う。

もっとも合理的な択を使えることと、相手のそれに対策できることが基本。
読み合いを考えてない相手にはそれで勝てるし、よっぽど相手が読み勝つのでなければそれが一番勝率も上がる。
問題は自分が最も合理的な択ばかりを選んだ場合に相手がそれに一点読みを合わせて裏をかいてくる場合。こちらは裏の裏にしないといけない。

まずは一番太い択を見せて、相手の反応を見る。

裏をかかないといけないのは相手の読みが強いときだけ。ただ、相手の読みがどの程度か判断するのは難しい。まずは裏をかくとまではいかなくとも直線的でなく、相手の読みが悪くても情勢が悪くはならない手から選ぶ。

手を選ぶ前に狙いがある。狙いと手が合ってないとリソースが無駄になりやすい。

駆け引きには読み合いも含むけど、その前に情勢の把握と、攻めと守りの切り替え。機会を伺うこと。

誰かの人生はゲームの景品じゃない。
日常生活でわざわざ駆け引きしたくない。

試し行動も駆け引き。
