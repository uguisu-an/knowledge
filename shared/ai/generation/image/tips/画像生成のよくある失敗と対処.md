よくある失敗と対処

| 症状                   | 原因                       | 対処                                                       |
| ---------------------- | -------------------------- | ---------------------------------------------------------- |
| 手足がおかしい         | SD全般の弱点               | ネガティブにbad hands, extra fingersを追加、画角に入れない |
| 顔が崩れる             | CFG高すぎ or Steps少なすぎ | CFGを下げる、Stepsを増やす                                 |
| 思ったスタイルが出ない | モデルが合っていない       | モデルを変える                                             |
| のっぺりした絵         | ライティング指定が弱い     | cinematic lighting, dramatic shadowsなど追加               |
| 毎回違いすぎる         | Seedがランダム             | 気に入ったSeedを固定                                       |
| 指示が混ざる           |                            |                                                            |

## ノイズが残る

1. Stepsを増やす (増やしすぎるのもよくない)
2. SamplerをDPM++系にする
3. テキストガイダンスを調整する
4. img2imgの場合、強度を上げる

## 手足がおかしい

1. ネガティブプロンプトで予防する
2. そもそも画角に入れない
3. 部分修正で直す

## 指示が混ざる

Attribute DriftやColor Bleedingと呼ばれる問題。
二つ以上の箇所に対する特徴や色の指示が混ざってしまう。
特に、SDXL系は指示の解析能力が低いので混ざりやすい。

性別や年齢層などで区別できるなら、区別できる情報をつけた方が確実。
例えば、

```
2girls,
while shirt woman, black skirt woman,
orange shirt girl, black shorts girl,
```

Reginal Prompterを導入する方法もある。
画像を複数のカラムに分割して、それぞれで強調する属性を変えるような方法。

## アニメ調にならない

ネガティブプロンプトでリアル系の出力を制限する。

```
photo, realistic, photorealistic,
```
