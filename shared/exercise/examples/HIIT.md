# High Intensity Interval Training: HIIT

高強度インターバルトレーニング

高負荷の短い運動と小休憩を繰り返すトレーニング方法。
高強度というのは基本的に無酸素運動の負荷が高い運動。

基本的に強度が高いので、しっかり準備運動してから行う。

決まった運動があるわけではないが、高強度の運動として代表的なのはバービージャンプやマウンテンクライマー。

運動20秒、休憩10秒を一連にして、4種目2セットで4分か、5種目2セットで5分がちょうどいい。

https://www.nas-club.co.jp/magazine/hiit/

HIITは心拍数を高めるのがポイント。ただ、初心者がいきなり心拍数を上げようとしてもそもそも体力が足りない。まずは息が上がって会話ができないくらいの運動と、長めの休憩から始める。

HIITで短い休憩を挟むのはまた次に動くとき全力で動くため。ただし、心拍数を下げないぎりぎりの時間で休む。休みすぎると心拍数が下がって効果が落ちてしまう。

高強度インターバルトレーニング。
短時間の高負荷運動と、短い休憩または低負荷運動を交互に繰り返す方法。

心拍数を上げて、脂肪燃焼効果と代謝アップが期待できる。
筋肉を大きくするよりは、体力をつけたり、脂肪を落としたりするタイプの筋トレ。

https://x.com/hosomacho1/status/2020020985651425322?s=12
https://www.s-re.jp/magazine/health/102/

以下の種目を運動20秒、休憩10秒の4種目で2分。これを2周して4分。
4分でランニングを30分続けたのと同じくらい心肺機能が向上したという研究もある。

1. バーピー
2. マウンテンクライマー
3. スクワット・スラスト
4. ジャンピング・ジャック

まずは1種目ずつ慣れていって、種目を増やしていくといい。

そもそも一つの種目をこなせないくらいの筋力だとできない。

朝起きていきなりHIITは重いので、最低でも30分くらいは経ってから準備運動して行うといい。

## 初心者向けの種目

- プッシュアップ (腕立て伏せ)
- もも上げ
- スクワットジャンプ: しゃがむ、立つ、ジャンプ、の繰り返し
- スクワットスラスト: しゃがむ、足を後ろへ伸ばす、戻す、立つ、の繰り返し
- マウンテンクライマー: 腕立ての姿勢でもも上げ
