# Burpee

基本的な流れ:

1. 立っている状態から始める
2. しゃがんで手を地面につける
3. 足を後ろに伸ばす
4. 足を戻してしゃがんだ状態に戻る
5. 立ち上がる

スクワットとハイプランクを合わせた状態とも言える。

立ち上がる際にジャンプしたり、さらにジャンプしながら腕を振り上げたり、プランクの際にプッシュアップしたり、といったバリエーションがある。
