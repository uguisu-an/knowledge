# 香りの系統

- 動物性
  - ムスク: ジャコウジカの香り
    - ホワイトムスク: 石鹸のような香り
    - ブラックムスク: 革のような香り
- 植物性
  - フルーティ: 果物
    - シトラス
      - レモン
      - オレンジ
      - マンダリン
      - ベルガモット
    - ライチ
    - ベリー
  - フローラル: 花
    - ローズ
    - ジャスミン
    - ラベンダー
  - ウッディ: 樹木
    - サンダルウッド（白檀）
  - グリーン: 草や葉
    - ハーバル: 薬草
    - モッシー: 苔
      - オークモス
    - クマリン: 桜餅の香り
  - スパイス: 香辛料
    - シナモン
    - カルダモン
- 複合系
  - オリエンタル: 甘い香り
    - バニラ
    - グルマン
  - フゼア（フジェール）: ラベンダー、オークモス、クマリンが主軸
  - マリン: 海の塩っぽさ
  - シプレ: フルーティ、フローラル、ウッディ、と変化する
  - アクアマリン: フルーティ、マリン、ムスク、と変化する

石鹸の香りはホワイトムスクが近い。
