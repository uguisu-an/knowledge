## エディター

- Visual Studio Code: 結局これ
- Obsidian: ノート同時を関連づける特殊なコンセプトのエディタ

## VSCodeの拡張機能

- Markdown All in One: 基本セット
- URL title resolver for Markdown: URLからタイトルを取得できる
- Markdown Preview Enhanced: LaTeXを描画できる
- Markdown Preview Mermaid Support: MermaidのUMLを描画できる
- Marp for VS Code: Marpをサポートする
- Markdown PDF: MarkdownをPDF印刷できる
- Markdown Preview Github Styling: GitHub風のスタイルをサポートする

## その他

- Marp: Markdownからスライドを生成する
- Pandoc: MarkdownからPDFやHTMLを生成する
