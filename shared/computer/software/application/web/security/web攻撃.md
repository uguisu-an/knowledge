## web攻撃の例

- インジェクション攻撃
  - SQLインジェクション
  - OSコマンドインジェクション
  - クロスサイトスクリプティング (XSS): 不正なスクリプトをwebページに注入して実行させる
    - リフレクト型
    - ストア型
- セッション攻撃
  - クロスサイトリクエストフォージェリ (CSRF)
  - パスワードリスト/ブルートフォース攻撃
  - セッションハイジャック
- 負荷攻撃
  - DoS攻撃
  - スローHTTP攻撃
- 改竄
- 踏み台 (SSRF)
- クリックジャッキング: 偽ボタンや偽リンクを配置してユーザーに操作させる

## web攻撃の対策

- 通信の暗号化 (HTTPS)
- セキュリティヘッダーの設定
- 入力値のバリデーション
