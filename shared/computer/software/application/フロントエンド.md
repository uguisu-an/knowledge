# フロントエンド

ユーザーが利用するシステムにおいて、ユーザー側の端のサービスを「フロントエンド」という。

関心としては UI サーバーやクライアントアプリケーションなど。

フロントエンドのコードを分けると、Vercel などのホスティングに任せられる利点もある。

フロントエンドやバックエンドはシステムの文脈の言葉。

フロントエンドの設計では、UIコンポーネントにはこだわるけど、データ型はUIデザインに合わせた方がやりやすい。
ただ、チームやデザイナーが優秀でDDDがわかるなら、ドメインモデルとUI上に配置される情報は近くなるはず。ユーザー視点で扱いやすいドメインモデルは情報設計として優れていないといけない。
