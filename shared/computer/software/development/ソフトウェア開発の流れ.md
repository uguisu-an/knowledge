企画などは製品開発の流れを踏襲する。

- 企画
  - 続行の判断
  - 使用する技術の選定
    - プログラミング言語
    - フレームワークやライブラリ
    - プラットフォーム
      - クラウドかオンプレか
      - 外部API
  - UXの検討
- 設計
  - 外観
    - UI
    - 外部に公開するAPI
  - システム
    - 機能
    - プラットフォーム
      - ネットワーク
        - API
  - モジュール
    - 命名
    - インタフェース
- 試験
  - プログラミング
  - コーディング
    - クラスやオブジェクト
    - 関数や変数
  - テスト
    - 単体テスト
    - 統合テスト
- 配信
  - デプロイ
  - 広告
- その他
  - ドキュメントの作成

開発作業は、設計、実装、試験、整理、反映、までセットで扱う。どれかが欠けていたらタスクは完了してない。
