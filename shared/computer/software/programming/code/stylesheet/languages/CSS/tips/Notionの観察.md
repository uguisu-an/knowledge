# Notion の観察

## タイムラインビュー

行ヘッダの幅や年月の表示位置は window 幅に合わせて細かく制御している。
バーの位置は absolute で制御している。
行ヘッダはコンテンツと別のボックスに入っていて、sticky-left で上に載せてる。これでヘッダ部分で左右にスクロールしてもコンテンツが動く。行の高さは固定して一致させている。
バーと文字が別々に描画されてて、文字だけは位置によって absolute と sticky を切り替えてるからバーとずらして表示できる。
バーに mask がかかっててバーの上にない文字は灰色になる。
バーの後ろに隠しブロックがあって、隠しブロックが表示されたところで移動用の矢印の左右を切り替えてる。
バーの左右に隠しブロックがあって、hover すると幅を変えるためのハンドルが表示される。
画面幅で高さ 0 の隠しブロック置いて padding でツールバーの表示位置制御してる。
