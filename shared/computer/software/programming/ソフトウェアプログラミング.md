プログラムを考えるだけじゃなくて、ソフトウェア製品としての設計や試験も必要。

- ファイルの配置
- ファイル名
- 関数、クラス、オブジェクトなどの名前
- 入力
- 出力
- など

後から変更しやすいようにする。

ソフトウェアサービスやアプリケーションを動かす、あるいはもっというとコンピュータを動かすという大きな括りの中にソフトウェアプログラミングがある。まずはコンピュータを動かすことを知らないといけない。

プログラミングやコーディングはソフトウェア開発のごく一部でしかない。

コンピュータに何らかの処理を実行してもらうまでの工程。

コンピュータプログラミングのことをそのままコーディングと呼ぶ場合がある。

Computer ProgrammingとCodingは同じと考えていい。
