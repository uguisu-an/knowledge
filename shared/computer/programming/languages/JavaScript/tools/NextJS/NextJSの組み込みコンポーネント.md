## Link

基本的に、内部リンクには `<Link>` を使う。
必要な差分だけ読み込んで切り替えてくれたり、プリフェッチが走ってローディングが速くなったりする。また、状態を保持したまま遷移できる。

以下の場合は例外として `<a>` を使った方がいい。

- 外部リンク
- アンカーリンク
- ファイルのダウンロード
- メールや電話 (`mailto:`や`tel:`)
- 強制リロードが必要な時

Nextのプリフェッチは軽量なので、ヘッダーのタイトルからトップページに戻るような動線も含めて `<Link>` でいい。
遷移先の画像が多すぎてプリフェッチしたくない場合は`<a>`にするよりも`<Link prefetch>`を設定する。
