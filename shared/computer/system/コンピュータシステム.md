個人の技量で何とかシステムを改修した場合、その人が抜けるとまたメンテナンスされなくなりがち。結局、組織でやるしかない。

情報通信、情報のやり取りが本質。

コンピュータシステムというのは、コンピュータネットワークのこと。仮想的なものも含む。
ただ、システム内の各サービスやアプリケーションの管理も含む。

コンピュータが集まったネットワーク内で、各ノードが情報の変換と通信を繰り返すことでシステムとして動いている。

ソフトウェアやアプリケーションだけでなく、ネットワークやハードウェアのことまで考える。まぁ、ハードウェアはクラウドなどで抽象化されることも多いけど、実際には個人の業務用PCからアクセスがあるし、後ろからモニターを見られてパスワードがバレるなどのアナログなセキュリティリスクもある。

システム開発に関わるもの:

- infrastructure
- network
- interface
- user
- computer

システム開発ってもはや計算機工学でもいいのかも。
