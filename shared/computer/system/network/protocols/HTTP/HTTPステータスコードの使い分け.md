- 200系: 正常
  - 201: 新しいデータを作った
  - 204: 本文がない
  - 200: その他
- 300系: リクエストに対して複数のレスポンスがある（主にリダイレクト）
  - 301: リソースが恒久的に移動した
  - 302: リソースが一時的に移動した
- 400系: クライアントエラー
  - 401: 認証されていない
  - 403: サーバーが要求を拒否した（主に権限エラー）
  - 404: 要求されたリソースが存在しない
  - 422: 要求の形式は合っているが、内容に誤りがある（主にバリデーションエラーやデータの競合）
  - 400: その他
- 500系: サーバーエラー
  - 503: 一時的に応答できない（主にメンテナンスや過負荷）
  - 500: その他

入力を変えたらなんとかなるエラーは400系、どうにもならない問題は500系にする。
