情報にアクセスできなくても後回しでいいなら多少止まってもいい。

個人開発でアクセスが少ないサービスなら固定費を抑えるために AWS Lambda もあり。
AWS Lambda ならアクセスが増えると高くなるけど、逆にアクセスがない時は全く費用がかからない。
ただし、アクセスが増えてくると変動費が重くなるので固定費に移行するのも大事。
