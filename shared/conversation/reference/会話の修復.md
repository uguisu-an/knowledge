# Repair

会話において「修復」と言われるプロセスは、誤解や聞き取りの失敗をどう処理するか、の部分。

## 修復が必要なとき

- 話し手が言い間違えた
- 聞き手が聞き取れなかった
- 話し手が誤解していた
- 聞き手が理解できなかった
- 話が逸れた
- 話が途切れた

聞き手が誤解した場合は聞き手が話し手になった時にわかるので、話し手が誤解していたパターンと同じ。

## 修復のスタイル

1. 自己開始自己修復: 話し手が気づいて、話し手が直す（「昨日、いやおとといスーパーに行ったんだけど」とか）
2. 他者開始自己修復: 聞き手が気づいて、話し手が直す（「田中さんが来るって」「田中さん？」「あ、鈴木さんだった」とか）
3. 自己開始他者修復: 話し手が気づいて、聞き手が直す（「あの、ほら、丸いやつ」「ボール？」とか）
4. 他者開始他者修復: 聞き手が気づいて、聞き手が直す（「来週の金曜、9日に」「金曜は10日だよ」とか）

修復のスタイルには優先順位がある。
自己修復が優先される。

自己開始と他者開始、自己修復と他者修復がある。

他者開始他者修復は面子を潰す行為なので、避けた方がいい。
他者開始自己修復みたいに変なところだけ指摘して自分で直してもらう。

修復開始のタイミング、会話の頑健性。

## 修復開始の表現

- 全体の繰り返し要求
  - え？
  - ん？
  - は？
  - なんて？
  - ごめん、もう一回言って
  - ごめん、聞こえなかった
- 特定箇所の繰り返し要求
  - ９日？
  - 何日って？
- 解説要求
  - どれ？
  - どういうこと？
  - さっき〜って言わなかった？
- 話を戻す
  - とにかく、さっきの話に戻すと、
  - 本題に戻ると、
  - その話で思い出したけど、
  - ところで、
  - 話が途切れちゃったね

聞き取れなかったのか、理解できなかったのか、でも使い分けられる。

非言語的な表現

- 首を傾げる
- 眉をひそめる
- 困惑した表情
- 手を耳に当てる仕草

## その他

どんなに流暢な会話でも修復は生じる。修復が全くない会話の方が不自然。

聞き返しが起こる要因:

- 内的要因:
  - 声が小さい
  - 滑舌が悪い
  - 曖昧な表現
    - 文脈が共有できていない
    - 代名詞や指示語が多すぎる
  - 話があちこちに飛んで追いにくい
- 外的要因:
  - 周囲の環境音
  - 聞き手の注意の揺らぎ
  - お互いの前提知識のずれ
