# 平和

争いがない。

安全、安心、安定を揃える。

安定のためには、リスクもあらかじめ排除しなければならない。
安心しきっているだけでは安定に繋がらない。

積極的平和主義

- 争いを起こさない
- 争いを起こされないように予防する
- 争いを起こされたら早急に対処する

消極的平和主義

- 争いを起こさない
- 争いを起こされやすい
- 争いを起こされても対処できない

好んで争いを起こさないとしても、争いを起こされたら対処しなければならない。

愛と優しさによって政治の必要性を弱められることがある。

平和は不自然。
平和を維持するには努力が必要だ。

争いがあるのが自然。

平和のためには、争いを起こす者に死んでもらったほうがいい。
殺すのではなく、死ぬのを見逃すのであれば平和は保たれる。
