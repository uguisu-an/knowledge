# 宗教

宗教は、Education + Engagementでできている。

宗教は教義という文化を教育する部分と、教会、つまりコミュニティの部分がある。

単なる文化は宗教ではないが、それを広めようとするのは宗教だ。

宗教の本質は教育すること。ただの文化ではなく、それを広めること。流行よりは伝統を重視する傾向にある。

「教えが間違っていたこともまた教え」

神を信じるなら、神が言ったのか、人が勝手に神の言ったことだと誤魔化して言ったのか、区別する必要がある。

宗教は哲学者にとってはいいおもちゃだけど、それ以外の人にとっては救いでもあるし、ただの習慣でもある。

キリスト教は自ら奴隷になる国民を育てるための国家宗教。

ユダヤ教は奴隷を慰めるための宗教。

多神教的な価値観からすると一神教は神の一柱がイキってるイキリ神なんよ。

宗教は教育の面が重要。民族や文化を形成する基礎になる。

独自の神を定めずに、宗教としての在り方を定めるだけの宗教があったっていい。

宗教はエンゲージメントと相性がいい。

神への信仰は、無私や喜捨と深い関わりがある。

単純に教育によって恩を売ってエンゲージメントを高める面もある。

日本の宗教観の基礎になっているのは神道、仏教、儒教あたり。儒教は特に朱子学。
三つとも世界や生命については言及してなくて、あと神もあまり考慮されない。どちらかというと人の意思を重視する。

家族や組織と同じように、宗教も関係を維持するために努力を必要とする。能動的に関係を維持することそのものが重要。

日本は宗教先進国すぎて、もはや宗教という形式すら残ってない。ただ慣習があるだけ。意識しなくても振る舞いの中に教えが馴染んでいる。
