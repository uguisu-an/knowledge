# Scale

一定の規則に従って音を並べたもの。

代表的な音階にメジャースケールとマイナースケールがある。

長調の音階 = メジャースケール
短調の音階 = マイナースケール

スケールを選んで使うのは、音と音の間隔をあえて不均等にして、それぞれの音に特徴を持たせるため。
https://www.logical-arts.jp/archives/2929

12音全てを使うのはクロマチックスケールという。

メジャースケールは明るく、マイナースケールは暗く聞こえる。

Cメジャースケールは、C（ド）を基点としてピアノの白鍵だけを全て使うのが特徴。
ドレミファソラシドの8音。
音と音の間隔で言うと、全、全、半、全、全、全、半。

Aマイナースケールは、A（ラ）を基点としてピアノの白鍵だけを全て使うのが特徴。
ラシドレミファソラの8音。
音と音の間隔で言うと、全、半、全、全、半、全、全。

モードは、ベースになるスケールからいくつかの音をフラットしたりシャープしたりして印象を変えたもの。
特にメジャー、マイナースケールがベースになったチャーチモードが有名。

スケールの一番目の音を主音という。CメジャーならC。

主音の一つ前が導音。スケールの七番目の音。
スケールの中で最も不安定な響きがあり、安定した主音を求める、導くことから導音という。
導音→主音の流れが曲の切れ目に使われやすい。導音で切れると中途半端な感じ。

音程が上がる時と下がるときで使う音が違うスケールがある。イ短調も。
