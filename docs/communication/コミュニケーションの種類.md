# コミュニケーションの種類

- 目的による違い
- 方法による違い
  - メディアの違い
  - チャネルの違い
  - 単方向・双方向
  - 言語的・非言語的
  - 形式的・非形式的（定型的・非定型的）

具体的には、

- Conversation: 言語的
- Discussion: 結論を出すためのやりとり
- Assertion: 自己主張
- Presentation: 形式的
- Education: 文化の伝達
- Documentation: 書類
- Narration: 物語
- Consultation: 相談
- Collaboration: 共同作業

Consultation は Discussion の中でも特に専門家との意見交換のこと。
