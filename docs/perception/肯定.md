# 肯定

「そうだね」「そうなんだ」「なるほど」とか。

「あなたの話を聞いている」という態度を示す。

肯定は同意とは違う。あなたの話を理解したよ、ということを伝える。

肯定しても賛同しなくていい。
賛同というのは Authority の問題。承認や権限。

物事を正しく認識しなければ、肯定しているというよりもわかっていないだけになる。

何もわかってないのに「うんうん、そうだね」と頷いている人は、詳しく掘り返されるとしどろもどろになってむしろ信頼を失う。

肯定というのは、否定しないこと。
フラットに、良くも悪くも判断しないで、まずは存在を認知すること。

否定というのは、認めないこと。目を背けることでもある。

肯定というのは認知すること、関心を持つことなのかも。

肯定的というのは、積極的に深く知ろうとすること、関心を持つこと。

肯定することと感謝することは似ている。

「あなたのやり方はまちがっている」というのは相手の勇気を挫いてしまう。

ポジティブは、楽観的とは違う。
あるものをあるがままに、否定しないこと。

とにかく肯定的に、存在を否定しないというのがあらゆる面で重要。「知らなくていい」という考えを捨てる。

「安倍さんはようやっとる」みたいな言葉は肯定的にも見えるけど、具体例を出せないなら、肯定しているのではなくて単に何も見てないだけ。ある意味で否定的とも言える。興味がない。

ポジティブなのは周りのためになる以上に自分のためになる。

存在を肯定するとして、例えば相手が犯罪者だったら普通に通報して現実のものとして扱うだけでいい。
愛しているなら味方になるけど、それでも罪を償うために自首を勧めるのか、一緒に罪を隠して捕まるのか、味方といっても対応は一つではない。

とにかく存在を否定せずに、受け入れるところから始める。
問題があれば現実的に対処する。

肯定するというのは、対処しないのとは別。

あるものはある。
無駄に否定したり排除したりしなくていい。

あるものをあるがままに認知する。判断しない。反応しない。

ないものではなく、あるものに目を向ける。

ネガティブなものから目を背けるというより、ありのままを見て肯定する。否定しない。

ネガティブなことから目を背けるのは愛がない。ただ、関心を広く持ちすぎるのは無理がある。身の丈にあった関心を持って、少しずつ広げよう。

ポジティブであるためにネガティブなことから目を背けるのは肯定しているとは言えない。
ただ、自分が悟りに至らないことを自覚して、過程としてとりあえずポジティブを目指すのは悪くない。
仏教でも世俗を離れて寺に入るものだし。
