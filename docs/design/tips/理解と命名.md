# 理解と命名

名前がないものは理解できていない。

正しい名前が正しい理解に繋がる。

違うものを同じ名前で呼ぶべきではない。

言葉は定義よりも「なぜ呼び分けられるようになったか」の方が重要。

名前をつけていないものは、理解できていないものである。
