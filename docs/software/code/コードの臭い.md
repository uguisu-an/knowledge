# Code Smell

コードの臭い、悪臭。

## Examples

- 重複したコード
- 長すぎるメソッド
- 巨大なクラス
- 長すぎるパラメータリスト
- 変更の偏り
- 変更の分散
- 特性の横恋慕
- データの群れ
- 基本データ型への執着
- Switch文
- パラレル継承
- 怠け者クラス
- 疑わしき一般化
- 一時的属性
- メッセージの連鎖
- 仲介人
- 不適切な関係
- クラスのインターフェース不一致
- 未熟なクラスライブラリ
- データクラス
- 相続拒否
- コメント
- 過剰な設計
- フラグ引数
