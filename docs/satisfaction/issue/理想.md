# 理想

理想は、叶えられなくても近づける。
理想を叶えられないからといって、近づくことすら諦めるのはやりすぎ。

誤差を縮めたら、縮めた分だけ幸せになれる。
時には理想の方から譲歩することもある。

理想が大きいのは、「遠い」だけ。

理想が遠ければ偉いわけでもない。

将来の理想を描かないと、目先の問題を潰すばかりになってしまう。

理想の姿を描けてないと「なぜそうなってないのか」が書けない。
理想の姿を描くためには、知識をより多く詰め込むしかない。

知識がないと、何番煎じかわからない陳腐な理想しか描けない。
あらゆる理想を知った上で、それを超えた理想を描く。

もしこれが実現したら、を夢想する。
