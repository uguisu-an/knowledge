- データソース
- データセット
- ビジュアル（分析）

## データソース

データソースは、ファイルやデータベースへの接続。

## データセット

データセットは、データソースから取り出した一部のデータ。
データベースにおけるビューみたいなもの。

表を結合したり、集計したり、値を変換したりできる。
データセット同士を結合することもできる。

## ビジュアル

ビジュアルは、データセットを表示する。

各種グラフやテーブルなどのテンプレートが用意されている。
値のフォーマットやデータのフィルターも設定できる。

## ダッシュボード

複数のビジュアルやフィルターを設置できる。
複数のシートに分けてタブで切り替えることもできる。
