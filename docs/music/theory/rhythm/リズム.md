# リズム

律動とも。

音の有無、つながり。あるいは長さやその変化。調子。
音と音の間隔、長さを並べたもの。
連続したり、休止したり。

特に音と音の間隔、速度を「テンポ」という。

途中でリズムやテンポが切り替わることもある。

数学的なアプローチ
感覚的なアプローチ

リズムがあるのは音楽というよりも「信号」の特徴。間隔がある。

テンポ感とビート感がある。

リズムは発音だけじゃなくて、休符の長さ（間）も重要。
