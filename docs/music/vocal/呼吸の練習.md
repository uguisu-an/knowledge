# 呼吸の練習

1. 姿勢を正す
2. 息を細く長く吐ききる (8 秒以上)
3. 息を吸う (5 秒)
4. 息を止める (5 秒)
5. 2 に戻る

まずは息を吐くのに 8 秒から始める。
最終的には 60 秒を目指して、20 秒、40 秒、60 秒とレベルアップする。
