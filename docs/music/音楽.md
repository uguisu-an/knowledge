# 音楽

「デザイン」というと視覚的なデザインが代表的だが、音などもデザインの範囲。
あらゆるインタフェースにデザインを考えられる。

音楽であれば、単音をリズム、メロディ、ハーモニーの三要素で曲として構成する理論がある。

学校の音楽科では、器楽、声楽、作曲、楽理などを習う。
