# 働き者と怠け者

「働き者」は行動先行タイプ、「怠け者」は思考先行タイプのことで、どちらが良いというものでもない。

有能な働き者は、前線の指揮官にする。
有能な怠け者は、将官にする。
無能な怠け者は、前線の兵士にする。
無能な働き者は、邪魔になるので処分するしかない。

これは仕事が兵士しかない場合の対策。
現実には、転職が第三の道になる。

働き者と怠け者の違いは着手が早いかどうか。
着手が早すぎると失敗や後戻りが多いし、遅すぎると機会を逃しやすい。

主体性を育てるのは難しい。
主体性がない場合は強制的に動かないといけない状況を作らないといけない。

無能な働き者とかはモルトケに由来しているらしい。
