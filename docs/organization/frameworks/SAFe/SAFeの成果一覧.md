## PI (Program Increment)

中期計画。三ヶ月くらいの周期で行うことが多い。
上位で決めた PI をベースにそれぞれのイテレーションを考える。

全体とスクラムのインクリメントの中間的なタイムボックス。
PI 単位で同期をとりながら開発を進める。

Program Backlog を作る。
