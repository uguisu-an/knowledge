# 科学

科学はプロセス。

科学の中心は、反証可能性。
そこから派生して再現性を生じている。

論理的に考えて、検証すること。

検証できる法則を見つけること。

反証されるたびに新しい法則を見つけていく。

「科学的」というのは「意識的」というのと近い。フィードバックによる学びを得て、偶然でなく必然に近づけようとする試みが科学だ。

科学の基本は、観察 → 疑問 → 仮説（予測）→ 検証の流れ。
特に、検証（反証）できる仮説を立てて検証するのが一番大事。

科学は神秘を常識に変える。

科学は法則を理解するところまで。利用するのは科学技術や工学。

道すじを示すのが科学で、そうでないのが奇跡や神秘。

知識を増やすプロセス。

科学の基礎は論理と検証。

科学は経験主義的。
実際の事象や実験を重視する。
