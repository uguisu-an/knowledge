# ニーズ

やるべきこと、課題。

ニーズは価値を生み出すために解消すべき課題。

欲しいものというよりは、特定の目的のために必要なもの。やるべきこと。目的主導。

ニーズの前に目的がある。
ニーズを満たすのは目的を達成するための手段。

ニーズを考える時は、
誰が、いつ、どういうことをしたいのか、
どのような障壁や阻害要因があるか、
今はどうやって解決していて、それでは何が困るか、
を考える。

問題がニーズになって、解決がゴールになる。

ニーズを満たす前にゴールを考える。
ゴールとニーズがずれていたら、本当のニーズではない。

問題自体は解決できてるけど手間や時間を省きたいのはニーズよりもワンツになりやすい。
