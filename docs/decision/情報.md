# 情報 (information)

未知を既知にするもの。

情報を集める → learning
情報を広める、与える → communication
情報を貯める → documentation
情報を捨てる → decision

情勢判断や意思決定に用いる。

知識や経験も、情報にしないと判断に使えない。

知識も経験も情報を構築するための素材でしかない。

知識があると暗黙的な条件を穴埋めできることがある。
知識がなければ、情報を集めないといけない。

なんらかの事実や意見があっても、問いがないと観察対象と結びつかない。
問いが事実や意見を情報にする。

データ、答えだけでは情報にならない。問いが必要。
