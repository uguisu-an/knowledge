データ分析を execution に応用する。

1. 達成したいことを明確にする
2. 解決したい課題を特定する
3. 課題を引き起こす要因を推測する
4. 仮説を実証するのに必要なデータを集める
5. 集めたデータを分析する
6. 目的から分析まで一貫しているか振り返る
7. 組織を動かす
8. アクションを実行する

データ分析の基本的な流れ

1. 分析の目的を明確にする
2. 分析の計画を立てる
3. データを収集する
4. データを加工する
5. 分析を実施する
6. 洞察を述べる
