# オムニチャネル Omnichannel

実店舗やアプリ、カタログ、ウェブ、EC サイトなど、顧客とのあらゆる接点（チャネル）で最適な購買体験を提供することで販売増を目指すマーケティング手法。

購入方法、受け取り、決済、商品の渡し方が複数用意されている仕組みをオムニチャネルという。アップルストアもそうだ。

オムニチャネルには、社内在庫の一元管理システムが必要。

クロスチャネルとオムニチャネルでいうと、オムニチャネルは複雑な組み合わせが特徴。

サプライチェーンマネジメントが必須になる。販売、物流センター、配送のプラットフォームが必要。

商品の受け取りも pub と sub がある。

商品を渡すところ、ラストワンマイルは顧客との接点になりうる。店舗の強みはここ。

オムニチャネルの最低条件

- 在庫情報の一元管理
- 価格の統一
- 店員教育

参入障壁でもある。
特に在庫情報の一元管理システムを構築しないといけないのが重い。

オムニチャネルは面白い考え方だけど、大きいとこ向けだな。
