# 契約の種類

契約によって、発注者、受注者、作業者、の三者の関係が異なる。

| 契約           | 指示系統         | 指示の変更 | 作業の上限 |
| -------------- | ---------------- | ---------- | ---------- |
| 派遣           | 発注者 -> 作業者 | できる     | 工数       |
| 委任・準委任   | 発注者 -> 受注者 | できる     | 工数       |
| 請負・業務委託 | 発注者 -> 受注者 | できない   | 完成       |
