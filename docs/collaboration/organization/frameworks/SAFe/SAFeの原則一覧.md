# Principles

1. 経済的な視点を持つ
2. システム思考を用いる
3. 変動を想定して、選択肢を用意しておく
4. 統合された迅速な学習サイクルを用いて、漸進的に構築する
5. マイルストーンは動作するシステムの客観的な評価に基づく
6. 進行中の作業を可視化して制限し、バッチサイズを減らし、待ち行列の長さを管理する
7. リズムを適用し、部門横断の計画作りで同期する
8. ナレッジワーカーの内発的な動機を解き放つ
9. 権限を委譲し、意思決定を分散する
10. 価値を中心に組織を作る

原則を守って、方法論をカスタマイズする。
