# ユーモアとエスプリ

ユーモアと対立する概念として「エスプリ」がある。
ウィットともいう。

愛情が根底にあるユーモアに対して、エスプリは批判が根底にあるもの、とされる。

ユーモアは相手のことを対等に見て、不完全なまま肯定するような態度。
対して、エスプリは相手のことを下に見て、否定するような態度。

「ウィットに富んだ一言」というと、相手からの攻撃に対して同じように攻撃するのではなく、相手がおかしな人のように扱って遠回しに馬鹿にするさまを指す。
皮肉のように悪意が元というよりは、相手が自分から悪いところに気づけるような振る舞い。ただ、下に見てはいる。
