# デズモンド・モリスの「親密さの十二段階」

1. 目から体 → 挨拶、名前を呼ぶ、自分の視界に相手が入った状態
2. 目から目 → アイコンタクト
3. 声から声 → 会話
4. 手から手 → 握手、手つなぎ
5. 腕から肩 → 腕や脚の接触、肩組み、腕組みなど
6. 腕から腰 → 腰を抱く、ハグなど
7. 口から口 → キス
8. 手から頭 → 頭をなでる
9. 手から体 → わきをくすぐる、首、尻や腿を触るなど、前戯の始まり
10. 口から胸 → 前戯
11. 手から性器 → 挿入を伴わないセックス、セックスの代替行為
12. 性交 → セックス

「手から手」や「腕から肩」のあたりはまだ知人でもする。
