# 戦争

戦争は手段でしかない。
利益を得たいだけなら戦争は時代遅れだ。
国際的な交流が盛んになった今、影響範囲が広すぎてリスクが高すぎる。

戦争は起こるもの。起こすものではない。
どれだけ緊張を緩和させようとしても、破綻するときは破綻する。

戦争は当事者以外にも波及する。
第三者が邪魔に思って、あるいは便乗して殺しにくることもある。

戦争もビジネス。
競合の解決方法の一つでしかないし、無駄にコストが高いので利益を得るのは難しい。
関係が切れてしまって長期的な利益を得るのも難しくなる。
利益を得るために、害を与える。

戦争は手段でしかない。侵略または防衛が目的。
侵略戦争は止めるべきだが、防衛戦争は必要。

政治や経済の失敗が戦争という形で現れる。

害を与えるのが戦争の本質。

アベレージを高める戦い方と、大一番を勢いで制する戦い方のどちらもできないと勝ちきれない。

守るか、奪うか、の軸がある。

思考や感情の表現がアートなら、それらの破壊が戦争。アイデンティティの否定。

防衛の目的は侵略の影響を受けないようにすること。
対抗手段としての戦争は持っておいて損はない。

戦争は手段でしかない。銃を捨てても刀で戦えるように、戦争を捨てても他の方法で侵略できる。

防衛戦争でも相手に害を与えることを目的とする。
侵略者は Win-Lose を狙っているが、最低でも Lose-Lose に持ち込む。
何も防衛できなければ、敵にもなれず、餌になるだけ。

競合や競争があると、戦争が起きやすい。
とはいえ、競争はなくなるものでもない。

「やられる前にやる」がうまくいくのは、やっても周りから攻められない時だけ。
複数のプレイヤーがいる場合、誰かを攻めても他から攻められたら損する可能性も高い。

相手の嫌がることをするのは戦争、敵への対処。

戦争は争いそのものではない。
争いを解決する手段の一つ。

戦争はどこかで終わりにしないと、それ自体を続けていても意味がない。

戦争では相手を滅ぼすのではなく、有利な講和を結んだ方が利になる。とはいえ、どっちみち戦争した時点でいつか反抗されるので、滅ぼしてしまった方が安全な気もする。
