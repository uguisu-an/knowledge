# おすすめの設定

- タッチパッドの速度を上げる
  （システム設定 > トラックパッド > 軌跡の速さ）

Macのデフォルトのキーボード設定は日本人向けじゃないので変更した方がいい。

- 英字入力中にスペルを自動変換: OFF
- 文頭を自動的に大文字にする: OFF
- 二重引用符: ""
- 一重引用符: ''

## おすすめのアプリケーション

- Alfred: ランチャー
- Homebrew: パッケージ管理
- Raycast: クリップボード拡張
- Google Chrome: ブラウザ
- Visual Studio Code: ソフトウェア開発、ドキュメンテーション
