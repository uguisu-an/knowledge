# 会計 Accounting

経理

経済的なトランザクションや財務情報の処理、記録、報告を行う。

ビジネスとトランザクション、財務状況や業績を正確に把握するための活動。

行政や市場に対して説明のために公開する会計は財務会計と呼ばれることがある。
