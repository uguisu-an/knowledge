# 物語

物語は変化を描くものだ。

「変化への抵抗」も面白みになる。

物語で子どもが中心になるのは「不安定」で「変わりやすい」からだ。

1. 起：変化する前
2. 承：変化のきっかけ
3. 転：変化の過程
4. 結：変化した後

変化への「圧力」と変化への「抵抗」のせめぎ合いも物語の見所である。
