基礎知識:

- ファイルの拡張子には ".go" を使う
- プログラムはパッケージで構成される
- プログラムは main パッケージから開始される
- main パッケージに main 関数があれば実行される
- パッケージ名はインポートパスの最後の要素と同じ名前になる、例えば math/rand なら rand になる
- 大文字で始まる名前はパッケージの外から参照できる (exported)
- 関数は複数の戻り値を返せる
- 関数の戻り値にも名前をつけられる

文法:

- 変数名の後ろに型名を書く
