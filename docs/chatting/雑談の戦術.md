# 雑談の戦術

## 普段から話題を集める

こないだ買ったアイスがいまいちだったとか、ちょっとしたことでいい。
ただし、相手にも関係あることにする。食事とか行楽とか。話題が切り替わっていくので、どうでもいい話題ほどいい。

語れるものはあるか？

好きなもののどんな部分が好きか？

## 話が尽きたら新しい話題を供給する

誰かに質問して話題を引き出すよりも、自分から新しい話題を提供する。
ただし、話題を提供しても、自分が話し過ぎない。相手が同じ話題で話し始めたら聞きに回る。

## 新しい話題が共有されたら乗り換える

自分の話題にこだわらない。
主題が移り変わっていくのが雑談の特徴。
