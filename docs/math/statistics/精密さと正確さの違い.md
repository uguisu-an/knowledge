# 精度と正確さ Precision vs Accuracy

精度は複数の値の間でのばらつきが小さいこと。

正確さはある値が狙った値に近いこと。

銃で的を撃つとすると、的から外れていても同じところに当たるのが精度で、ばらついても中心に近いのが正確さ。
