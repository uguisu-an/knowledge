1. 原始関数を知っていたら変換する
2. 原始関数を知らなかったら知っている形に変換する
3. そもそも解けない形なら問題設定を変更して解ける形で研究する

まずはこれ。
積分は原始関数を知らないと解けない。
原始関数は研究されたものが存在する。

知っている形に変換するテクニックとしては、置換積分、部分積分、King Property がある。
