# Exclusive / Disjoint

「排反である」とは、二つの事象が同時に起こることはないということ。

2 つの事象 A, B の積が空事象（集合的にいうと互いに素）である場合、A と B は互いに排反である、という。

$$
A \cap B = \varnothing
$$

二つの事象が排反であるならば、$P(A \cap B) = 0$ である。
逆にいうと、 $P(A \cap B) = 0$ でなければ、事象 A と B は排反ではない。

ただし、同時に起こる確率がゼロであっても排反であるとは限らない。
排反でなくても $P(A \cap B) = 0$ になることはある。

公理より、互いに排反の事象の確率の和は単に足し合わせればいい。
排反でなければ、共通部分の確率を引かなければならない。

$$
P(A \cup B) = P(A) + P(B)
$$
