二つの確率変数 X, Y の和も確率変数になる。

$$X + Y$$

離散確率変数を例にすると、次のような表になる。

| X/Y   | $x_1$                      | $x_2$                      | ... |
| ----- | -------------------------- | -------------------------- | --- |
| $y_1$ | $(x_1 + y_1) \cdot p_{11}$ | $(x_2 + y_1) \cdot p_{21}$ | ... |
| $y_2$ | $(x_1 + y_2) \cdot p_{12}$ | $(x_2 + y_2) \cdot p_{22}$ | ... |
| ...   | ...                        | ...                        |     |

以上の表からもわかるように、独立した二つの確率変数の和の期待値は、

$$
E(X + Y) = \sum_{j, k}{(x_j + y_k) P(x_j, y_k)}
$$
