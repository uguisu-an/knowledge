# CDF: Comulative Distribution Function

$$累積分布関数 \Rightarrow 確率分布$$

単に分布関数とも。

確率変数 X の実現値がある値 x 以下になる確率を表す関数のこと。
確率密度関数 $f(x)$ と区別するために、大文字で $F(x)$ のように表す。

$$F(x) = P(X \le x)$$

特に連続型確率変数では負の無限大から x までの確率密度関数を定積分したもの。

「出る目が４以下となる確率」のようにある範囲の確率を求める場合に使う。
