# 機械学習 Machine Learning

機械になんらかのモデルを学習させて、特定用途の出力を得られるようにする。
「判断を下す」ということもある。

入力の処理方法を機械に覚えさせる。

さらに深層学習などの手法がある。

機械学習のエンジンに対して、特定用途に特化した学習済みのモデルを販売するビジネスもある

やっていることは基本的にプログラミングと同じ。関数＝モデルを作る。
機械が関数を生成している。

機械としては「学習」をしていて、その結果として「思考」ができる機械が作れる。

機械学習は、入力と結果から逆算して独自の計算機を作っている。逆向きにも作れる。
結果を整理して、何が入力されたのか区別するとか。踊るメイドインワリオもこの技術で作られているはず。特に人間が作ると条件分岐が細かすぎて時間がかかるものを機械に作らせる。
