# Stable Diffusion

画像生成の AI。

Stable Diffusion では学習済みモデルを checkpoint ファイルとして入出力できる。

Stable Diffusion Web UI でブラウザから操作できる。

Stable Diffusion のオンライン版は学習済みモデル使えないので、カスタマイズしたかったら基本的にはインストールが必要。
