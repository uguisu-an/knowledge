# Digital Transformation: DX

「デジタル革命」の意味。
これ自体は営業用の言葉で、とにかくコンピュータの活用を進めること。

ただ、IPA の「推進スキル標準」で示される以下の役割を特に育成しないといけない方針はわかりやすい。

1. ビジネスアーキテクト
2. デザイナー
3. データサイエンティスト
4. ソフトウェアエンジニア
5. サイバーセキュリティ
