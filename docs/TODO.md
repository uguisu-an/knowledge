## 概念ツリーの補完候補

現状のツリーに存在しない、または弱い上位概念。知識が蓄積したタイミングで `/retree` で反映を検討する。

### Self / Identity（自己・アイデンティティ）

`attention` / `devotion` / `hope` などが Human>Mind に散在しているが、「自己とは何か」「アイデンティティ」「自己認識」を束ねる上位概念がない。

### Space / Environment（空間・場所・環境）

`home` / `going` / `transportation` / `earth` が別々のカテゴリに散在している。「空間・場所・環境」という軸で束ねる可能性がある。

### Value / Ethics 統合軸

`value`（Work）と `ethics`（Society）が分離したまま。「価値・規範・倫理」を統合する概念軸の検討余地がある。

### Time の横断性

現在 Work 直下に置いているが、World/Life/Human/Work すべてを横断する基礎概念でもある。独立した扱いが適切かもしれない。
