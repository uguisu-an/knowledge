## ツェッテルカステン

ツェッテルカステンは、マインドマップに近い。
ノートをノードにマインドマップをしているイメージ。

1/1/2 みたいなのもノートの id で、ネストした id をつけるのは近しいものを近くに置くための手段。
ファイルで管理するなら、全部一つのフォルダに入れて、1_1_2.md みたいにするだけでできる。

Markdown なら保存するときに hook でリンク更新できるといいかも。
