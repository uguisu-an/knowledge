# Knowledge

個人的なナレッジをまとめたもの。

ナレッジそのものというよりも、領域の区分が特に重要。
ネット上で検索できる細かい知識については検索方法やURLに譲っていい。

ノートは `shared/`（中核）・`apps/`・`games/` に置く。
リポジトリ自体の構成・書き方・整理方針・ツールは [docs/](docs/) にまとめている。

- 概念マップ → [docs/map.md](docs/map.md)
- ノートの書き方 → [docs/conventions.md](docs/conventions.md)
- 整理の方針 → [docs/organization.md](docs/organization.md)
- 構成・フォルダ索引 → [docs/folder-index.md](docs/folder-index.md)
- 整形・ツール → [docs/tooling.md](docs/tooling.md)
