# 景気の方向

| 名称               | 物価 | 賃金 | 景気 |
| ------------------ | ---- | ---- | ---- |
| インフレーション   | ▲    | ▲    | 好   |
| デフレーション     | ▽    | ▽    | 悪   |
| スタグフレーション | ▲    | ▽    | 悪   |

給料は上がってないのに物価が上がるスタグフレーションはデフレとインフレの悪いとこどり。

デフレは不景気を長引かせる。デフレは実質的に借金の支払いが増えるので、なだらかなインフレが良いとされている。
デフレはデフレを呼ぶ。デフレスパイラル。物価が下がれば企業の売上も支払える給料も減って、さらに需要が減る。消費者がものを買えなくなる。さらに物価が下がる。

物価が急に高くなると通貨の価値が下がりすぎて、相対的には生活が苦しくなる。インフレというよりはインフレ率が重要な指標。
また、消費者が急いで買い集めようとすると生産が追いつかなくなって、さらに供給の不足 → 物価の上昇が起きる。
